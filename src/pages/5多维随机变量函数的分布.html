【方法1】多维→一维：
已知 $(X,Y) \sim p_{ij}$ 或 $f(x,y)$，求 $Z = g(X,Y)$ 的分布律或概率密度
【方法2】一维→多维：
已知 $X$ 的分布，构造新的多维随机变量 $(U,V)$
【方法3】多维→多维：
已知 $(X,Y)$ 的分布，求 $(U,V)$ 的联合分布

一、多维→一维 (Ok(II任目标1))

1. (离散型，离散型)→离散型 (Dk(常规操作))

基本思路

(1) 若 $(X,Y) \sim p_{ij}$，$Z = g(X,Y)$，有 $Z \sim q_l$

(2) 若 $X \sim p_i$，$Y \sim q_k$，$X, Y$ 独立且取值在某一集合，可考虑 $Z = X + Y$，$XY$，$\max\{X,Y\}$，$\min\{X,Y\}$ 等

重要公式：独立情况下的分布

① 若 $Z = X + Y$，且 $X, Y$ 独立并取非负整数，则

$$P\{Z = k\} = P\{X + Y = k\}$$

D2k(卷积等价表述)： $$= P\{X = 0\}P\{Y = k\} + P\{X = 1\}P\{Y = k-1\} + \cdots + P\{X = k\}P\{Y = 0\}$$ $$= p_0q_k + p_1q_{k-1} + \cdots + p_kq_0, \quad k = 0, 1, 2, \cdots$$

② 若 $Z = \max\{X,Y\}$，且 $X, Y$ 独立并取非负整数，则

$$P\{Z = k\} = P\{\max\{X,Y\} = k\}$$

D2k(卷积等价表述)： $$= P\{X = k, Y = k\} + P\{X = k, Y = k-1\} + \cdots + P\{X = k, Y = 0\} +$$ $$P\{X = k-1, Y = k\} + P\{X = k-2, Y = k\} + \cdots + P\{X = 0, Y = k\}$$ $$= p_kq_k + p_kq_{k-1} + \cdots + p_kq_0 + p_{k-1}q_k + p_{k-2}q_k + \cdots + p_0q_k$$ $$k = 0, 1, 2, \cdots$$

③ 若 $Z = \min\{X,Y\}$，且 $X, Y$ 独立，$0 \leq X, Y \leq l$，$l$ 为正整数，$X, Y$ 取整数时，则

$$P\{Z = k\} = P\{\min\{X,Y\} = k\}$$

D2k(卷积等价表述)： $$= P\{X = k, Y = k\} + P\{X = k, Y = k+1\} + \cdots + P\{X = k, Y = l\} +$$ $$P\{X = k+1, Y = k\} + P\{X = k+2, Y = k\} + \cdots + P\{X = l, Y = k\}$$ $$= p_kq_k + p_kq_{k+1} + \cdots + p_kq_l + p_{k+1}q_k + p_{k+2}q_k + \cdots + p_lq_k$$ $$k = 0, 1, 2, \cdots, l$$

📌 例 5.1

题目：设随机变量 $X$ 与 $Y$ 相互独立，且 $X$ 服从参数为 $\lambda$ 的泊松分布，

$$Y \sim \begin{pmatrix} -1 & 1 \\ \frac{1}{4} & \frac{3}{4} \end{pmatrix}$$

求 $Z = XY$ 的概率分布。

查看详细解答

【解】已知 $P\{X = k\} = \frac{\lambda^k}{k!}e^{-\lambda}$ $(k = 0, 1, \cdots)$，$Y$ 可能取值为 $-1, 1$，$X$ 与 $Y$ 相互独立，故 $Z = XY$ 可能取值为 $0, \pm 1, \pm 2, \cdots, \pm k, \cdots$，其概率分布为

$$P\{Z = XY = 0\} = P\{X = 0\} = e^{-\lambda}$$ $$P\{Z = XY = k\} = P\{X = k, Y = 1\} = P\{X = k\}P\{Y = 1\} = \frac{3\lambda^k}{4k!}e^{-\lambda}, \quad k = 1, 2, 3, \cdots$$ $$P\{Z = XY = -k\} = P\{X = k\}P\{Y = -1\} = \frac{1\lambda^k}{4k!}e^{-\lambda}, \quad k = 1, 2, 3, \cdots$$

2. (连续型，连续型)→连续型

三种方法

(1) 分布函数法 (Dk(常规操作))

设 $(X,Y) \sim f(x,y)$，$Z = g(X,Y)$，则

$$F_Z(z) = P\{g(X,Y) \leq z\} = \iint_{g(x,y)\leq z} f(x,y)\,\mathrm{d}x\mathrm{d}y$$ $$f_Z(z) = F'_Z(z)$$

(2) 换元法 (D2k(观察研究对象))

设 $(X,Y) \sim f(x,y)$，若 $\begin{cases} u = u(x,y) \\ v = v(x,y) \end{cases}$ 具有一阶连续偏导数，且存在唯一的反函数 $\begin{cases} x = x(u,v) \\ y = y(u,v) \end{cases}$，又

$$J = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \\ \frac{\partial u}{\partial u} & \frac{\partial v}{\partial v} \end{vmatrix} \neq 0$$

则 $(X,Y)$ 的函数 $\begin{cases} U = U(X,Y) \\ V = V(X,Y) \end{cases}$ 的联合概率密度为 $f_{U,V}(u,v) = f[x(u,v), y(u,v)]|J|$

$U$ 的概率密度为 $f_U(u) = \int_{-\infty}^{+\infty} f_{U,V}(u,v)\,\mathrm{d}v$，$V$ 的概率密度为 $f_V(v) = \int_{-\infty}^{+\infty} f_{U,V}(u,v)\,\mathrm{d}u$

【注】(1) 证明 $(U,V)$ 的联合分布函数为 $$F_{U,V}(u,v) = P\{U \leq u, V \leq v\} = P\{U(X,Y) \leq u, V(X,Y) \leq v\}$$ $$= \iint_{U(x,y)\leq u \atop V(x,y)\leq v} f_{U,V}(x,y)\,\mathrm{d}x\mathrm{d}y = \iint_{U(x,y)\leq u \atop V(x,y)\leq v} f[x(u,v), y(u,v)]|J|\,\mathrm{d}u\mathrm{d}v$$

故 $f_{U,V}(u,v) = \frac{\partial^2 F}{\partial u \partial v} = f[x(u,v), y(u,v)]|J|$

(2) 在高等数学中，我们已经熟练掌握了二重积分的换元积分法，故在此给出换元法求 $f(u,v)$，这是十分有用且简便的方法。

(3) 最值函数的分布 (D2k(观察研究对象))

① $\max\{X,Y\}$ 的分布

设 $(X,Y) \sim F(x,y)$，则 $Z = \max\{X,Y\}$ 的分布函数为

$$F_{\max}(z) = P\{\max\{X,Y\} \leq z\} = P\{X \leq z, Y \leq z\} = F(z,z)$$

当 $X$ 与 $Y$ 相互独立时，

$$F_{\max}(z) = F_X(z) \cdot F_Y(z)$$

② $\min\{X,Y\}$ 的分布

设 $(X,Y) \sim F(x,y)$，则 $Z = \min\{X,Y\}$ 的分布函数为

$$F_{\min}(z) = P\{\min\{X,Y\} \leq z\} = P\{\{X \leq z\} \cup \{Y \leq z\}\}$$ $$= P\{X \leq z\} + P\{Y \leq z\} - P\{X \leq z, Y \leq z\}$$ $$= F_X(z) + F_Y(z) - F(z,z)$$

当 $X$ 与 $Y$ 相互独立时，

$$F_{\min}(z) = F_X(z) + F_Y(z) - F_X(z)F_Y(z) = 1 - [1 - F_X(z)][1 - F_Y(z)]$$

推广到 $n$ 个相互独立的随机变量 $X_1, X_2, \cdots, X_n$ 的情况，即

$$F_{\max}(z) = F_{X_1}(z)F_{X_2}(z)\cdots F_{X_n}(z)$$ $$F_{\min}(z) = 1 - [1 - F_{X_1}(z)][1 - F_{X_2}(z)]\cdots[1 - F_{X_n}(z)]$$

特别地，当 $X_i$ $(i = 1,2,\cdots,n)$ 相互独立且具有相同的分布函数 $F(x)$ 与概率密度 $f(x)$ 时，

$$F_{\max}(z) = [F(z)]^n, \quad f_{\max}(z) = n[F(z)]^{n-1}f(z)$$ $$F_{\min}(z) = 1 - [1 - F(z)]^n, \quad f_{\min}(z) = n[1 - F(z)]^{n-1}f(z)$$
📌 例 5.2

题目：设随机变量 $X, Y$ 相互独立，且均服从参数为 $\lambda$ 的指数分布，令 $Z = |X - Y|$，则下列随机变量中与 $Z$ 同分布的是（  ）。

(A) $X + Y$     (B) $\frac{X + Y}{2}$     (C) $2X$     (D) $X$

查看详细解答

【解】应选 (D)。

$X$ 与 $Y$ 的联合概率密度为 $f(x,y) = f_X(x) \cdot f_Y(y) = \begin{cases} \lambda^2 e^{-\lambda(x+y)}, & x > 0, y > 0 \\ 0, & \text{其他} \end{cases}$

设 $Z$ 的分布函数为 $F_Z(z)$，则 $F_Z(z) = P\{Z \leq z\} = P\{|X - Y| \leq z\}$

当 $z < 0$ 时，$F_Z(z) = 0$；

当 $z \geq 0$ 时，

$$F_Z(z) = P\{-z \leq X - Y \leq z\} = 2P\{0 \leq X - Y \leq z\}$$ $$= 2\int_0^{+\infty} \lambda e^{-\lambda x}\,\mathrm{d}y \int_y^{y+z} \lambda e^{-\lambda x}\,\mathrm{d}x$$ $$= 2\int_0^{+\infty} \lambda e^{-\lambda y}[e^{-\lambda y} - e^{-\lambda(y+z)}]\,\mathrm{d}y$$ $$= 2\int_0^{+\infty} \lambda e^{-2\lambda y}\,\mathrm{d}y - 2e^{-\lambda z}\int_0^{+\infty} \lambda e^{-2\lambda y}\,\mathrm{d}y$$ $$= 1 - e^{-\lambda z}$$

所以 $Z \sim E(\lambda)$，从而 $Z$ 与 $X$ 服从相同分布，选 (D)。

📌 例 5.3

题目：设 $(X,Y) \sim f(x,y) = \begin{cases} 8xy, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$，记 $Z = \frac{X}{Y}$。

(1) 求 $Z$ 的概率密度 $f_Z(z)$；

(2) 求 $P\left\{Y < \frac{2}{3}\Big| X = \frac{1}{2}\right\}$。

查看详细解答

【解】(1) 令 $\begin{cases} u = \frac{x}{y} \\ v = y \end{cases}$，即有 $\begin{cases} x = uv \\ y = v \end{cases}$，$J = \begin{vmatrix} v & u \\ 0 & 1 \end{vmatrix} = v$，于是 $f_{U,V}(u,v) = \begin{cases} 8uv \cdot v \cdot v, & 0 < uv < 1 \\ 0, & \text{其他} \end{cases} = \begin{cases} 8uv^3\,\mathrm{d}v, & 0 < u < 1 \\ 0, & \text{其他} \end{cases} = \begin{cases} 2u, & 0 < u < 1 \\ 0, & \text{其他} \end{cases}$

故 $f_U(u) = \int_{-\infty}^{+\infty} f_{U,V}(u,v)\,\mathrm{d}v = \begin{cases} \int_0^1 v^3\,\mathrm{d}v, & 0 < u < 1 \\ 0, & \text{其他} \end{cases} = \begin{cases} 2u, & 0 < u < 1 \\ 0, & \text{其他} \end{cases}$

故 $f_Z(z) = \begin{cases} 2z, & 0 < z < 1 \\ 0, & \text{其他} \end{cases}$

(2) 由于 $P\left\{X = \frac{1}{2}\right\} = 0$，故不能直接用条件概率公式，

$$f_X(x) = \int_{-\infty}^{+\infty} 8xy\,\mathrm{d}y = \begin{cases} \int_x^1 8xy\,\mathrm{d}y, & 0 < x < 1 \\ 0, & \text{其他} \end{cases} = \begin{cases} 4x(1-x^2), & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

故

$$f_{Y|X}\left(y\Big|x = \frac{1}{2}\right) = \frac{f\left(\frac{1}{2},y\right)}{f_X\left(\frac{1}{2}\right)} = \frac{4y}{\frac{3}{2}} = \frac{8}{3}y, \quad \frac{1}{2} < y < 1$$

于是 $P\left\{Y < \frac{2}{3}\Big| X = \frac{1}{2}\right\} = \int_{\frac{1}{2}}^{\frac{2}{3}} y\,\mathrm{d}y = \frac{8}{3} \times \frac{1}{2} \times \left(\frac{4}{9} - \frac{1}{4}\right) = \frac{7}{27}$

📌 例 5.4

题目：设随机变量 $X_i$ 与 $X_j$ 相互独立且同分布，其概率密度为

$$f(x) = \begin{cases} ax, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

(1) 求 $a$ 的值；

(2) 求 $Z = \max\{X_1, X_2\} - \min\{X_1, X_2\}$ 的概率密度。

查看详细解答

【解】(1) 由归一性，$\int_{-\infty}^{+\infty} f(x)\,\mathrm{d}x = \int_0^1 ax\,\mathrm{d}x = \frac{1}{2}a = 1$，做 $a = 2$

D2k(化归程典型式)

(2) $Z = \max\{X_1, X_2\} - \min\{X_1, X_2\} = \frac{X_1 + X_2 + |X_1 - X_2|}{2} - \frac{X_1 + X_2 - |X_1 - X_2|}{2} = |X_1 - X_2|$

令 $\begin{cases} x_1 - x_2 = u \\ x_2 = v \end{cases}$，即 $\begin{cases} x_1 = u + v \\ x_2 = v \end{cases}$，则 $J = \begin{vmatrix} 1 & 1 \\ 0 & 1 \end{vmatrix} = 1$，且 $\begin{cases} 0 < u + v < 1 \\ 0 < v < 1 \end{cases}$

当 $0 < u + v < 1$，且 $0 < v < 1$ 时（见图），有

$$f_{U,V}(u,v) = f_X(u+v) \cdot f_Y(v) = 2(u+v) \cdot 2v$$

即有

$$f_U(u) = \int_{-\infty}^{+\infty} f_{U,V}(u,v)\,\mathrm{d}v$$

当 $0 < u < 1$ 时，$Z = |U|$ 的分布函数为

$$F_Z(z) = P\{|U| \leq z\} = P\{-z \leq U \leq z\}$$

故当 $0 < z < 1$ 时，$Z$ 的概率密度为

$$f_Z(z) = f_U(z) + f_U(-z) = \frac{2}{3}z^3 - 2z + \frac{4}{3} + \left(-\frac{2}{3}z^3\right) + 2(-z) + \frac{4}{3} = \frac{4}{3}z^3 - 4z + \frac{8}{3}$$

于是，$Z$ 的概率密度为

$$f_Z(z) = \begin{cases} \frac{4}{3}z^3 - 4z + \frac{8}{3}, & 0 < z < 1 \\ 0, & \text{其他} \end{cases}$$
📌 例 5.5

题目：设随机变量 $X \sim U(1,2)$，在 $X = x$ 的条件下，$Y$ 的条件分布为 $E(x)$，则 $D(XY)$ = (  )。

(A) 1     (B) 2     (C) $x$     (D) $2x$

查看详细解答

【解】应选 (A)。

设 $f(x,y) = f_X(x) \cdot f_{Y|X}(y|x) = 1 \cdot xe^{-xy}, 1 < x < 2, y > 0$

令 $\begin{cases} u = xy \\ v = y \end{cases}$，得 $J = \begin{vmatrix} \frac{u}{v} & -\frac{u}{v^2} \\ v & 1 \end{vmatrix} = \frac{1}{v}$，故

$$f_{U,V}(u,v) = \begin{cases} \frac{u}{v}e^{-u} \cdot \frac{1}{v}, & 1 < \frac{u}{v} < 2, v > 0 \\ 0, & \text{其他} \end{cases}$$

即有

$$f_U(u) = \int_{-\infty}^{+\infty} f_{U,V}(u,v)\,\mathrm{d}v$$ $$= \begin{cases} \int_{\frac{u}{2}}^u ue^{-u} \cdot \frac{1}{v^2}\,\mathrm{d}v, & u > 0 \\ 0, & \text{其他} \end{cases} = \begin{cases} e^{-u}, & u > 0 \\ 0, & \text{其他} \end{cases}$$

故 $XY$ 服从 $E(1)$，于是 $D(XY) = 1$。

3. (离散型，连续型)→连续型

① $\max\{X,Y\}$ 的分布

设 $(X,Y) \sim F(x,y)$，则 $Z = \max\{X,Y\}$ 的分布函数为

$$F_{\max}(z) = P\{\max\{X,Y\} \leq z\} = P\{X \leq z, Y \leq z\} = F(z,z)$$

当 $X$ 与 $Y$ 相互独立时，

$$F_{\max}(z) = F_X(z) \cdot F_Y(z)$$

② $\min\{X,Y\}$ 的分布

设 $(X,Y) \sim F(x,y)$，则 $Z = \min\{X,Y\}$ 的分布函数为

$$F_{\min}(z) = P\{\min\{X,Y\} \leq z\} = P\{\{X \leq z\} \cup \{Y \leq z\}\}$$
📌 例 5.6

题目：设总体 $X$ 服从 $[0,\theta]$ 上的均匀分布，$\theta$ 为正常数，$X_1, X_2, X_3$ 是取自 $X$ 的一个样本，求 $Y = \max\{X_1, X_2, X_3\}$，$Z = \min\{X_1, X_2, X_3\}$ 的分布函数和概率密度。

查看详细解答

【解】设 $F(x)$ 为 $X$ 的分布函数，$f(x)$ 为 $X$ 的概率密度，则

$$F(x) = \begin{cases} 1, & x \geq \theta \\ \frac{x}{\theta}, & 0 \leq x < \theta \\ 0, & x < 0 \end{cases}, \quad f(x) = \begin{cases} \frac{1}{\theta}, & 0 \leq x < \theta \\ 0, & \text{其他} \end{cases}$$

由 $Y = \max\{X_i\}$，$Z = \min\{X_i\}$，则 $Y$ 的分布函数与概率密度分别为

$$F_Y(y) = [F(y)]^3 = \begin{cases} 1, & y \geq \theta \\ \left(\frac{y}{\theta}\right)^3, & 0 \leq y < \theta \\ 0, & y < 0 \end{cases}$$ $$f_Y(y; \theta) = 3[F(y)]^2 f(y) = \begin{cases} 3\left(\frac{y}{\theta}\right)^2 \cdot \frac{1}{\theta}, & 0 \leq y < \theta \\ 0, & \text{其他} \end{cases}$$

$Z$ 的分布函数与概率密度分别为

$$F_Z(z) = 1 - [1 - F(z)]^3 = \begin{cases} 1, & z \geq \theta \\ 1 - \left(1 - \frac{z}{\theta}\right)^3, & 0 \leq z < \theta \\ 0, & z < 0 \end{cases}$$ $$f_Z(z; \theta) = 3[1 - F(z)]^2 f(z) = \begin{cases} 3\left(1 - \frac{z}{\theta}\right)^2 \cdot \frac{1}{\theta}, & 0 \leq z < \theta \\ 0, & \text{其他} \end{cases}$$

（离散型、离散型）往往人为制造，恍恍惚惚计数某事

2. 连续型→(离散型，离散型) (Dk(常规操作))

$$X \sim f(x), \quad \begin{cases} U = g(X) \\ V = h(X) \end{cases} \Rightarrow (U,V) \sim p_{ij}$$

三、多维→多维 (Ok(II任目标3))

1. (离散型，离散型)→(离散型，离散型) (Dk(常规操作))

$$(X,Y) \sim p_{ij}, \quad \begin{cases} U = g(X,Y) \\ V = h(X,Y) \end{cases} \Rightarrow (U,V) \sim q_{ij}$$

2. (连续型，连续型)→(离散型，离散型) 或 (连续型，连续型) (Dk(常规操作))

$$(X,Y) \sim f(x,y), \quad \begin{cases} U = g(X,Y) \\ V = h(X,Y) \end{cases} \Rightarrow (U,V) \sim p_{ij} \text{或} f_{U,V}(u,v)$$

3. (离散型，连续型)→(离散型，离散型) (Dk(常规操作))

$$X \sim p_i, \quad Y \sim f_Y(y), \quad \begin{cases} U = g(X,Y) \\ V = h(X,Y) \end{cases} \Rightarrow (U,V) \sim q_{ij}$$
📌 例 5.7

题目：设随机变量 $X_1, X_2, X_3$ 相互独立，其中 $X_1$ 与 $X_2$ 均服从标准正态分布，$X_3$ 的概率分布为

$$P\{X_3 = 0\} = P\{X_3 = 1\} = \frac{1}{2}, \quad Y = X_1X_3 + (1-X_3)X_2$$

(1) 求二维随机变量 $(X_1, Y)$ 的分布函数，结果用标准正态分布函数 $\Phi(x)$ 表示；

(2) 证明随机变量 $Y$ 服从标准正态分布。

查看详细解答

【解】(1) 记 $(X_1, Y)$ 的分布函数为 $F(x,y)$，则对任意实数 $x$ 和 $y$，都有

$$F(x,y) = P\{X_1 \leq x, Y \leq y\}$$

D2k(化归程典型式)

$$= P\{X_1 \leq x, X_1X_3 + (1-X_3)X_2 \leq y, \Omega\}$$ $$= P\{X_1 \leq x, X_1X_3 + (1-X_3)X_2 \leq y, \{X_3 = 0\} \cup \{X_3 = 1\}\}$$ $$= P\{X_1 \leq x, X_1X_3 + (1-X_3)X_2 \leq y\} P\{X_3 = 0\} +$$ $$P\{X_1 \leq x, X_1X_3 + (1-X_3)X_2 \leq y | X_3 = 1\}$$ $$= \frac{1}{2}P\{X_1 \leq x, X_2 \leq y | X_3 = 0\} + \frac{1}{2}P\{X_1 \leq x, X_1 \leq y | X_3 = 1\}$$ $$= \frac{1}{2}P\{X_1 \leq x, X_2 \leq y\} + \frac{1}{2}P\{X_1 \leq x, X_1 \leq y\}$$ $$= \frac{1}{2}P\{X_1 \leq x\}\Phi(y) + \frac{1}{2}P\{X_1 \leq \min\{x,y\}\}$$ $$= \frac{1}{2}\Phi(x)\Phi(y) + \frac{1}{2}\Phi(\min\{x,y\})$$

(2) 【证】由(1)知，$Y$ 的分布函数为

$$F_Y(y) = \lim_{x \to +\infty} F(x,y)$$ $$= \lim_{x \to +\infty} \left[\frac{1}{2}\Phi(x)\Phi(y) + \frac{1}{2}\Phi(\min\{x,y\})\right]$$ $$= \frac{1}{2}\Phi(y) + \frac{1}{2}\Phi(y) = \Phi(y)$$

所以 $Y$ 服从标准正态分布。

📌 例 5.8

题目：设随机变量 $X$ 满足 $|X| \leq 1, P\{X = -1\} = P\{X = 1\} = \frac{1}{4}$，当 $|X| < 1$ 时，$X$ 服从 $(-1,1)$ 上的均匀分布，$Y$ 服从参数为 1 的指数分布，$X, Y$ 独立，令 $Z = X + Y$，求：

(1) $X$ 的分布函数 $F_X(x)$；

(2) $Z$ 的分布函数 $F_Z(z)$。

查看详细解答

【解】(1) 当 $x < -1$ 时，$F(x) = P\{X \leq x\} = 0$；

当 $x = -1$ 时，$F(x) = P\{X \leq x\} = \frac{1}{4}$；

当 $-1 < x < 1$ 时，

D2k(化归程典型式)

$$F(x) = P\{X \leq -1\} + P\{-1 < X \leq x\}$$ $$P\{-1 < X \leq x\} = P\{-1 < X \leq x, \Omega\}$$ $$= P\{-1 < X \leq x, -1 < X < 1\} + P\{-1 < X \leq x, -1 < X < 1\}$$ $$= P\{-1 < X \leq x | -1 < X < 1\}P\{-1 < X < 1\} + P\{-1 < X \leq x | -1 < X < 1\}P\{-1 < X < 1\}$$ $$= \frac{x+1}{1-(-1)} \cdot \frac{1}{2} + 0 = \frac{x+1}{4}$$

故 $F(x) = \frac{1}{4} + \frac{x+1}{4} = \frac{2+x}{4}$

当 $x \geq 1$ 时，$F(x) = 1$

综上，$X$ 的分布函数为

$$F_X(x) = \begin{cases} 0, & x < -1 \\ \frac{2+x}{4}, & -1 \leq x < 1 \\ 1, & x \geq 1 \end{cases}$$

(2) 记 $A = \{-1 < X < 1\}$，由(1)知

$$f_X(x) = f_X(x | A)P(A) = \begin{cases} \frac{1}{4}, & -1 < x < 1 \\ 0, & \text{其他} \end{cases}, \quad \left(\text{或} f_X(x) = F'_X(x) = \begin{cases} \frac{1}{4}, & -1 < x < 1 \\ 0, & \text{其他} \end{cases}\right)$$ $$F_Z(z) = P\{Z \leq z\} = P\{X + Y \leq z\}$$

D2k(化归程典型式)

$$= P\{X = -1\}P\{Y \leq z+1\} + P\{X = 1\}P\{Y \leq z-1\} + P\{X + Y \leq z, -1 < X < 1\}$$

其中

$$P\{Y \leq z+1\} = F_Y(z+1) = \begin{cases} 1 - e^{-(z+1)}, & z+1 \geq 0 \\ 0, & \text{其他} \end{cases}$$ $$P\{Y \leq z-1\} = F_Y(z-1) = \begin{cases} 1 - e^{-(z-1)}, & z-1 \geq 0 \\ 0, & \text{其他} \end{cases}$$

① 当 $z < -1$ 时，$P\{X + Y \leq z, -1 < X < 1\} = 0$

② 当 $-1 \leq z < 1$ 时，

$$P\{X + Y \leq z, -1 < X < 1\} = \int_{-1}^z \mathrm{d}x \int_0^{z-x} \frac{1}{4}e^{-y}\,\mathrm{d}y = \frac{1}{4}\int_{-1}^z (1-e^{z-x})\,\mathrm{d}x$$ $$= \frac{1}{4}\int_{-1}^z (1-e^{z-x})\,\mathrm{d}x = \frac{1}{4}(z+1) - \frac{1}{4}e^{z-x}\Big|_{-1}^z$$ $$= \frac{1}{4}(z+1) - \frac{1}{4}(1-e^{-1-z})$$ $$= \frac{1}{4}(z + e^{-1-z})$$

③ 当 $z \geq 1$ 时，

$$P\{X + Y \leq z, -1 < X < 1\} = \int_{-1}^1 \mathrm{d}x \int_0^{z-x} \frac{1}{4}e^{-y}\,\mathrm{d}y$$ $$= \frac{1}{4}\int_{-1}^1 (1-e^{z-x})\,\mathrm{d}x$$ $$= \frac{1}{2} - \frac{1}{4}e^{1-z}\Big|_{-1}^1$$ $$= \frac{1}{2} - \frac{1}{4}(e^{1-z} - e^{-1-z})$$

综上，

$$F_Z(z) = \begin{cases} 0, & z < -1 \\ \frac{1}{4}[1 - e^{-(z+1)}] + \frac{1}{4}[z + e^{-(z+1)}], & -1 \leq z < 1 \\ \frac{1}{4}[1 - e^{-(z+1)}] + \frac{1}{4}[1 - e^{-(z-1)}] + \frac{1}{2} - \frac{1}{4}(e^{1-z} - e^{-1-z}), & z \geq 1 \end{cases}$$ $$= \begin{cases} 0, & z < -1 \\ \frac{1+z}{4}, & -1 \leq z < 1 \\ 1 - \frac{1}{2}e^{1-z}, & z \geq 1 \end{cases}$$
📌 例 5.9
