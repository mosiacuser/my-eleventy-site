📐 一、微分中值定理

费马(Fermat)引理

费马引理

设函数 $f(x)$ 在点 $x_0$ 处可导，如果函数 $f(x)$ 在点 $x_0$ 处取得极值，那么 $f'(x_0) = 0$。

罗尔(Rolle)定理

罗尔定理

定理内容：如果函数 $f(x)$ 满足以下条件：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导
  3. $f(a) = f(b)$

则在 $(a,b)$ 内至少存在一点 $\xi$，使得 $f'(\xi) = 0$。

几何意义：曲线上至少存在一点的切线平行于x轴。
拉格朗日(Lagrange)中值定理

拉格朗日中值定理

定理内容：如果函数 $f(x)$ 满足以下条件：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导

则在 $(a,b)$ 内至少存在一点 $\xi$，使得：

$$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$

推论

如果在 $(a,b)$ 内恒有 $f'(x) = 0$，则在 $[a,b]$ 上 $f(x)$ 为常数。

柯西(Cauchy)中值定理

柯西中值定理

定理内容：如果函数 $f(x)$，$F(x)$ 满足以下条件：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导，且 $F'(x)$ 在 $(a,b)$ 内每一点处均不为零

则在 $(a,b)$ 内至少存在一点 $\xi$，使得：

$$\frac{f(b) - f(a)}{F(b) - F(a)} = \frac{f'(\xi)}{F'(\xi)}$$

泰勒(Taylor)定理

带皮亚诺余项的泰勒公式

如果函数 $f(x)$ 在点 $x_0$ 处有直至 $n$ 阶的导数，则有：

$$f(x) = f(x_0) + f'(x_0)(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + R_n(x)$$

其中 $R_n(x) = o[(x-x_0)^n]$ 为皮亚诺余项。

带拉格朗日余项的泰勒公式

如果函数 $f(x)$ 在含有 $x_0$ 的开区间 $(a,b)$ 内有直至 $n+1$ 阶的导数，则当 $x \in (a,b)$ 时有：

$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$$

这里 $\xi$ 介于 $x_0$ 与 $x$ 之间，称为拉格朗日余项。

常用的麦克劳林公式（$x_0 = 0$）

1. 指数函数：

$$e^x = 1 + x + \frac{x^2}{2!} + \cdots + \frac{x^n}{n!} + \frac{e^\theta x}{(n+1)!}x^{n+1}, \quad (0 < \theta < 1)$$

2. 正弦函数：

$$\sin x = x - \frac{x^3}{3!} + \cdots + (-1)^n\frac{x^{2n-1}}{(2n-1)!} + (-1)^n\frac{\cos(\theta x)}{(2n+1)!}x^{2n+1}$$

3. 余弦函数：

$$\cos x = 1 - \frac{x^2}{2!} + \cdots + (-1)^n\frac{x^{2n}}{(2n)!} + (-1)^{n+1}\frac{\cos(\theta x)}{(2n+2)!}x^{2n+2}$$

4. 对数函数：

$$\ln(1+x) = x - \frac{x^2}{2} + \cdots + (-1)^n\frac{x^n}{n} + (-1)^n\frac{x^{n+1}}{(n+1)(1+\theta x)^{n+1}}$$

5. 幂函数：

$$(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \cdots + \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}x^n + \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)(\alpha-n)}{(n+1)!}(1+\theta x)^{\alpha-n-1}x^{n+1}$$

🔍 二、导数应用

1. 函数的单调性

定理

设函数 $y = f(x)$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内可导。

  1. 若在 $(a,b)$ 内 $f'(x) > 0$，则 $f(x)$ 在 $[a,b]$ 上单调递增
  2. 若在 $(a,b)$ 内 $f'(x) < 0$，则 $f(x)$ 在 $[a,b]$ 上单调递减

2. 函数的极值

极值的定义

设 $y = f(x)$ 在点 $x_0$ 及其邻域内有定义。如果对于该邻域内任何 $x$ 恒有：

  • $f(x_0) \geq f(x)$ 或 $f(x_0) > f(x)$，则称 $f(x_0)$ 为函数的极大值
  • $f(x_0) \leq f(x)$ 或 $f(x_0) < f(x)$，则称 $f(x_0)$ 为函数的极小值

极大值和极小值统称为极值。导数为零的点称为函数的驻点。

极值的必要条件

若 $y = f(x)$ 在点 $x_0$ 处可导，如果 $x_0$ 是 $f(x)$ 的极值点，则必有 $f'(x_0) = 0$。

极值的第一充分条件

设 $y = f(x)$ 在点 $x_0$ 的某邻域内可导，且 $f'(x_0) = 0$（或 $f(x)$ 在 $x_0$ 处连续）。

  1. 若 $x < x_0$ 时 $f'(x) > 0$，$x > x_0$ 时 $f'(x) < 0$，则 $f(x_0)$ 为极大值
  2. 若 $x < x_0$ 时 $f'(x) < 0$，$x > x_0$ 时 $f'(x) > 0$，则 $f(x_0)$ 为极小值
  3. 若 $f'(x)$ 在 $x_0$ 两侧同号，则 $f(x_0)$ 不为极值

极值的第二充分条件

设 $y = f(x)$ 在点 $x_0$ 处二阶可导，且 $f'(x_0) = 0$。

  1. 若 $f''(x_0) < 0$，则 $f(x_0)$ 为极大值
  2. 若 $f''(x_0) > 0$，则 $f(x_0)$ 为极小值
  3. 若 $f''(x_0) = 0$，则此方法失效，需用第一充分条件判断

3. 函数的最大值与最小值

最值问题求解步骤

问题1：求连续函数 $f(x)$ 在 $[a,b]$ 上的最大最小值

  1. 求出 $f(x)$ 在开区间 $(a,b)$ 的驻点和不可导点 $x_1, x_2, \ldots, x_n$
  2. 计算 $f(x)$ 在点 $x_1, x_2, \ldots, x_n$ 和区间端点 $a, b$ 处的函数值
  3. 比较以上各点函数值，其中最大者即为 $f(x)$ 在 $[a,b]$ 上的最大值，最小者即为最小值
注意：当连续函数 $f(x)$ 在 $[a,b]$ 内仅有唯一驻点 $x_0$，若 $x_0$ 为极大值或极小值点，则它也是 $f(x)$ 在 $[a,b]$ 上的最大值或最小值点。

4. 曲线的凹凸性

凹凸性定义

函数 $f(x)$ 在区间 $I$ 上连续，如果对 $I$ 上任意两点 $x_1, x_2$ 恒有：

$$f\left(\frac{x_1 + x_2}{2}\right) < \frac{f(x_1) + f(x_2)}{2}$$

则称 $f(x)$ 在 $I$ 上的图形是凹的（或称上凸）。

如果恒有：

$$f\left(\frac{x_1 + x_2}{2}\right) > \frac{f(x_1) + f(x_2)}{2}$$

则称 $f(x)$ 在 $I$ 上的图形是凸的（或称下凸）。

凹凸性判定定理

设 $y = f(x)$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内二阶可导，则：

  1. 若在 $(a,b)$ 内 $f''(x) > 0$，则 $f(x)$ 在 $[a,b]$ 上的图形是凹的
  2. 若在 $(a,b)$ 内 $f''(x) < 0$，则 $f(x)$ 在 $[a,b]$ 上的图形是凸的

拐点

连续曲线上凹弧与凸弧的分界点称为曲线的拐点。

拐点的必要条件：若 $(x_0, f(x_0))$ 为拐点，则必有 $f''(x_0) = 0$。

拐点的第一充分条件：设 $y = f(x)$ 在点 $x_0$ 的某邻域内二阶可导，且 $f''(x_0) = 0$（或 $f''(x)$ 在 $x_0$ 处连续）。若 $f''(x)$ 在 $x_0$ 左右两侧异号，则 $(x_0, f(x_0))$ 为 $y = f(x)$ 的拐点。

5. 曲线的渐近线

渐近线类型

水平渐近线：若 $\lim_{x \to \infty} f(x) = A$（或 $\lim_{x \to -\infty} f(x) = A$ 或 $\lim_{x \to +\infty} f(x) = A$），则 $y = A$ 是曲线 $y = f(x)$ 的水平渐近线。

铅直渐近线：若 $\lim_{x \to x_0} f(x) = \infty$（或 $\lim_{x \to x_0^-} f(x) = \infty$ 或 $\lim_{x \to x_0^+} f(x) = \infty$），则 $x = x_0$ 是曲线 $y = f(x)$ 的铅直渐近线。

斜渐近线：若 $\lim_{x \to \infty} \frac{f(x)}{x} = a$ 且 $\lim_{x \to \infty} [f(x) - ax] = b$（或 $x \to -\infty$ 或 $x \to +\infty$），则 $y = ax + b$ 是曲线 $y = f(x)$ 的斜渐近线。

6. 函数的作图

利用函数的单调性、极值、曲线的凹凸性、拐点及渐近线可以做出函数曲线的草图。

7. 曲线的弧微分与曲率

弧微分

设 $y = f(x)$ 在 $(a,b)$ 内有连续导数，则有弧微分：

$$ds = \sqrt{1 + (y')^2}dx$$

曲率

设 $y = f(x)$ 有二阶导数，则有曲率：

$$K = \frac{|y''|}{(1 + (y')^2)^{3/2}}$$

曲率半径 $\rho = \frac{1}{K}$。

8. 导数在经济学中的应用

经济学中常见的函数

1. 需求函数：$x = \varphi(p)$，其中 $x$ 为某产品的需求量，$p$ 为价格。

需求函数的反函数 $p = \varphi^{-1}(x)$ 称为价格函数。

2. 供给函数：$x = \psi(p)$，其中 $x$ 为某产品的供给量，$p$ 为价格。

3. 成本函数：成本 $C = C(x)$ 是生产 $x$ 件产品的总投入，由固定成本 $C_1$ 和可变成本 $C_2(x)$ 组成：

$$C = C(x) = C_1 + C_2(x)$$

平均成本：$\overline{C} = \frac{C}{x} = \frac{C_1}{x} + \frac{C_2(x)}{x}$

4. 收益函数：$R = R(x)$ 是产品售出后所得的收入，售单价为 $p$：

$$R = R(x) = px$$

5. 利润函数：$L = L(x)$ 是收益减去成本后的总收益：

$$L = L(x) = R(x) - C(x)$$

边际与弹性

边际概念

边际：对于可导的经济学函数 $y = f(x)$，$f'(x_0)$ 表示 $f(x)$ 在 $x = x_0$ 处的边际值。

常用的边际：

  • 边际成本：$MC = C'(q)$
  • 边际收益：$MR = R'(q)$
  • 边际利润：$ML = L'(q)$

弹性概念

弹性：对于函数 $y = f(x)$，称

$$\eta = \lim_{\Delta x \to 0} \frac{\Delta y / y}{\Delta x / x} = \frac{dy}{dx} \cdot \frac{x}{y} = f'(x) \cdot \frac{x}{f(x)}$$

为函数 $f(x)$ 在 $x$ 处的弹性，记为 $E_y$。

需求的价格弹性：设需求函数 $Q = \varphi(P)$（$P$ 为价格），则需求对价格的弹性为：

$$\eta_d = -\frac{P}{Q}\varphi'(P)$$

由于 $\varphi'(P) < 0$，所以 $\eta_d > 0$。

经济学意义：当价格上升（或下降）1%时，则需求量将减少（或增加）$|\eta_d|$%。

供给的价格弹性：设供给函数 $Q = \psi(P)$（$P$ 为价格），则供给的弹性为：

$$\eta_s = \frac{P}{Q}\psi'(P)$$

由于供给函数 $\psi'(P) > 0$，所以 $\eta_s > 0$。

例题 1：极值与最值问题

题目：在半径为 $R$ 的球中内接一直圆锥，求圆锥的最大体积。

解答：设圆锥的高为 $h$，底面半径为 $r$，则体积为：

$$V = \frac{1}{3}\pi r^2 h$$

由几何关系：$r^2 + (h-R)^2 = R^2$

因此：$r^2 = 2Rh - h^2$

代入体积公式：$V(h) = \frac{1}{3}\pi(2Rh - h^2)h = \frac{\pi}{3}(2Rh^2 - h^3)$

求导：$V'(h) = \frac{\pi}{3}(4Rh - 3h^2)$

令 $V'(h) = 0$，得 $h = \frac{4R}{3}$ 或 $h = 0$（舍去）

经验证，$h = \frac{4R}{3}$ 为极大值点，故最大体积为：

$$V_{\max} = \frac{32\pi R^3}{81}$$

例题 2：渐近线问题

题目：求曲线 $y = x\left(1 + \arcsin\frac{2}{x}\right)$ 的斜渐近线方程。

解答：设斜渐近线为 $y = ax + b$，则：

$$a = \lim_{x \to \infty} \frac{y}{x} = \lim_{x \to \infty} \left(1 + \arcsin\frac{2}{x}\right) = 1$$

$$b = \lim_{x \to \infty} (y - ax) = \lim_{x \to \infty} x \cdot \arcsin\frac{2}{x} = \lim_{x \to \infty} \frac{\arcsin\frac{2}{x}}{\frac{1}{x}}$$

使用洛必达法则或等价无穷小（$\arcsin t \sim t$ 当 $t \to 0$）：

$$b = \lim_{x \to \infty} \frac{\frac{2}{x}}{\frac{1}{x}} = 2$$

因此，斜渐近线方程为：$y = x + 2$

例题 3：不等式证明

题目：证明：当 $x > 0$ 时，$\frac{x}{1+x} < \ln(1+x) < x$。

证明：由拉格朗日中值定理可知：

$$\ln(1+x) = \ln(1+x) - \ln 1 = \frac{(1+\xi) - 1}{1+\xi} = \frac{x}{1+\xi}$$

其中 $0 < \xi < x$，因此 $1 < 1+\xi < 1+x$。

则：$$\frac{x}{1+x} < \frac{x}{1+\xi} < x$$

即：$$\frac{x}{1+x} < \ln(1+x) < x$$

例题 4：中值定理证明

题目：设 $f(x)$ 在闭区间 $[a,b]$ 上连续，在 $(a,b)$ 上二阶可导，且 $f(a) = f(b)$，$f(a) = f(c)$（$a < c < b$）。证明：存在 $\xi \in (a,b)$，使得 $f''(\xi) = 0$。

证明：由题设条件及罗尔定理知，存在 $\xi_1 \in (a,c)$，$\xi_2 \in (c,b)$，使得：

$$f'(\xi_1) = 0, \quad f'(\xi_2) = 0$$

在区间 $[\xi_1, \xi_2]$ 上对 $f'(x)$ 应用罗尔定理，存在 $\xi \in (\xi_1, \xi_2) \subset (a,b)$，使得：

$$f''(\xi) = 0$$

原题得证。
