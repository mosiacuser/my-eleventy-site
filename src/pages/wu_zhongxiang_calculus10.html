📚 第十章：无穷级数 - 交互式教程

常数项级数 幂级数 傅里叶级数

第一节 常数项级数

📖 定义：级数的概念

设\(\{u_n\}\)是一数列，则表达式

$$\sum_{n=1}^{\infty}u_n = u_1 + u_2 + \cdots + u_n + \cdots$$

称为无穷级数，简称级数。\(S_n = \sum_{i=1}^{n}u_i\)称为级数的部分和。

若部分和\(\{S_n\}\)有极限\(S\)，即\(\lim_{n\to\infty}S_n = S\)，则称级数收敛，称这个极限\(S\)为级数的和。

如果极限\(\lim_{n\to\infty}S_n\)不存在，则称级数发散。

1. 级数的性质

📐 性质1：线性性质

(1) 若级数\(\sum_{n=1}^{\infty}u_n\)收敛于\(S\)，则\(\sum_{n=1}^{\infty}ku_n\)也收敛，且其和为\(kS\)。

(2) 若\(\sum_{n=1}^{\infty}u_n\)和\(\sum_{n=1}^{\infty}v_n\)分别收敛于\(S\)、\(T\)，则\(\sum_{n=1}^{\infty}(u_n \pm v_n)\)收敛于\(S \pm T\)。

在级数中去掉、加上或改变有限项，不会改变级数的敛散性。一个级数的敛散性与其前有限项无关。
📐 性质2：收敛的必要条件

若级数\(\sum_{n=1}^{\infty}u_n\)收敛，则\(\lim_{n\to\infty}u_n = 0\)。

注意：\(\lim_{n\to\infty}u_n = 0\)不一定收敛（必要不充分）。
反之，若\(\lim_{n\to\infty}u_n \neq 0\)，则级数必发散。

例题：判断下列级数的敛散性

(1) \(\sum_{n=1}^{\infty}\ln\left(1+\frac{1}{n}\right)\)

(2) \(\sum_{n=0}^{\infty}aq^n\) (\(a \neq 0\))

解：

(1) \(S_n = \ln(1+1) + \ln(1+\frac{1}{2}) + \cdots + \ln(1+\frac{1}{n})\)

\(= \ln 2 + \ln\frac{3}{2} + \cdots + \ln\frac{n+1}{n} = \ln(n+1)\)

由于\(\lim_{n\to\infty}S_n = \lim_{n\to\infty}\ln(n+1) = \infty\)，故级数发散。

(2) 这是几何级数（等比级数）：

$$S_n = a + aq + aq^2 + \cdots + aq^{n-1} = \begin{cases} \frac{a(1-q^n)}{1-q}, & q \neq 1 \\ na, & q = 1 \end{cases}$$

当\(|q| < 1\)时，\(\lim_{n\to\infty}S_n = \frac{a}{1-q}\)，级数收敛；

当\(|q| \geq 1\)时，级数发散。

2. 正项级数的审敛准则

📖 定义：正项级数

若\(\sum_{n=1}^{\infty}u_n\)中\(u_n \geq 0\)，则称其为正项级数。

基本定理：正项级数\(\sum_{n=1}^{\infty}u_n\)收敛 \(\Leftrightarrow\) \(\{S_n\}\)有上界。

📐 比较判别法

设\(u_n \leq v_n\)，则：

  • \(\sum v_n\)收敛 \(\Rightarrow\) \(\sum u_n\)收敛
  • \(\sum u_n\)发散 \(\Rightarrow\) \(\sum v_n\)发散

比较判别法的极限形式：设\(\lim_{n\to\infty}\frac{u_n}{v_n} = l\) (\(0 \leq l < +\infty\))

  • 若\(0 < l < +\infty\)，则\(\sum u_n\)与\(\sum v_n\)同敛散
  • 若\(l = 0\)，\(\sum v_n\)收敛\(\Rightarrow\)\(\sum u_n\)收敛
  • 若\(l = +\infty\)，\(\sum v_n\)发散\(\Rightarrow\)\(\sum u_n\)发散
最常用的比较标准是p级数和等比级数：
$$\sum_{n=1}^{\infty}\frac{1}{n^p} \begin{cases} \text{收敛}, & p > 1 \\ \text{发散}, & p \leq 1 \end{cases}$$ $$\sum_{n=0}^{\infty}aq^n \begin{cases} \text{收敛}, & |q| < 1 \\ \text{发散}, & |q| \geq 1 \end{cases}$$
📐 比值判别法（d'Alembert判别法）

设\(\lim_{n\to\infty}\frac{u_{n+1}}{u_n} = \rho\)，则：

  • \(\rho < 1\) 时，级数收敛
  • \(\rho > 1\) 时，级数发散
  • \(\rho = 1\) 时，不能判定
📐 根值判别法（Cauchy判别法）

设\(\lim_{n\to\infty}\sqrt[n]{u_n} = \rho\)，则：

  • \(\rho < 1\) 时，级数收敛
  • \(\rho > 1\) 时，级数发散
  • \(\rho = 1\) 时，不能判定

3. 交错级数

📖 定义：交错级数

形如\(\sum_{n=1}^{\infty}(-1)^{n-1}u_n\)（其中\(u_n > 0\)）的级数称为交错级数。

📐 莱布尼茨判别法

若：(1) \(\{u_n\}\)单调减；(2) \(\lim_{n\to\infty}u_n = 0\)

则交错级数\(\sum_{n=1}^{\infty}(-1)^{n-1}u_n\)收敛。

4. 绝对收敛与条件收敛

📖 定义

绝对收敛：若级数\(\sum_{n=1}^{\infty}|u_n|\)收敛，则称\(\sum_{n=1}^{\infty}u_n\)绝对收敛。

条件收敛：若\(\sum_{n=1}^{\infty}u_n\)收敛，但\(\sum_{n=1}^{\infty}|u_n|\)发散，则称\(\sum_{n=1}^{\infty}u_n\)条件收敛。

绝对收敛的级数一定收敛，但收敛的级数不一定绝对收敛。

第二节 幂级数

📖 定义：幂级数

形如下式的函数项级数称为幂级数：

$$\sum_{n=0}^{\infty}a_nx^n = a_0 + a_1x + a_2x^2 + \cdots + a_nx^n + \cdots$$

或更一般地：

$$\sum_{n=0}^{\infty}a_n(x-x_0)^n = a_0 + a_1(x-x_0) + \cdots + a_n(x-x_0)^n + \cdots$$

1. 阿贝尔定理与收敛域

📐 阿贝尔(Abel)定理

(1) 若\(\sum_{n=0}^{\infty}a_nx^n\)在\(x=x_0\) (\(x_0 \neq 0\))处收敛，则对所有\(|x| < |x_0|\)，级数绝对收敛。

(2) 若\(\sum_{n=0}^{\infty}a_nx^n\)在\(x=x_0\)处发散，则对所有\(|x| > |x_0|\)，级数发散。

📖 收敛半径与收敛域

幂级数\(\sum_{n=0}^{\infty}a_nx^n\)的收敛性有且仅有以下三种情况之一：

  1. 对任意\(x \in (-\infty, +\infty)\)都收敛
  2. 仅在\(x=0\)处收敛
  3. 存在一个正数\(R\)，当\(|x| < R\)时绝对收敛，当\(|x| > R\)时发散

这个正数\(R\)称为幂级数的收敛半径，开区间\((-R, R)\)称为收敛区间。

求收敛半径的公式：

如果\(\lim_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right| = \rho\)，则\(R = \frac{1}{\rho}\)

如果\(\lim_{n\to\infty}\sqrt[n]{|a_n|} = \rho\)，则\(R = \frac{1}{\rho}\)

在求收敛域时，还需单独考察\(x = \pm R\)处的敛散性。收敛域是收敛区间加上收敛的端点。

例题：求幂级数\(\sum_{n=1}^{\infty}\frac{e^n-(-1)^n}{n^2}x^n\)的收敛半径和收敛域。

解：

$$\lim_{n\to\infty}\left|\frac{a_{n+1}}{a_n}\right| = \lim_{n\to\infty}\frac{e^{n+1}-(-1)^{n+1}}{(n+1)^2} \cdot \frac{n^2}{e^n-(-1)^n}$$

$$= \lim_{n\to\infty}\frac{e \cdot e^n + (-1)^n}{e^n - (-1)^n} \cdot \frac{n^2}{(n+1)^2} = e$$

因此，\(R = \frac{1}{e}\)，收敛区间为\((-\frac{1}{e}, \frac{1}{e})\)。

当\(x = \frac{1}{e}\)时，原级数为\(\sum_{n=1}^{\infty}\frac{1}{n^2}\)，收敛。

当\(x = -\frac{1}{e}\)时，原级数为\(\sum_{n=1}^{\infty}\frac{(-1)^n}{n^2}\)，绝对收敛。

收敛域为：\([-\frac{1}{e}, \frac{1}{e}]\)

2. 幂级数的性质

📐 性质1：连续性

设幂级数\(\sum_{n=0}^{\infty}a_nx^n\)的收敛半径为\(R\)，和函数为\(S(x)\)，则\(S(x)\)在收敛域上连续。

📐 性质2：可导性

\(S(x)\)在收敛区间\((-R, R)\)内可导，且可逐项求导：

$$S'(x) = \left(\sum_{n=0}^{\infty}a_nx^n\right)' = \sum_{n=0}^{\infty}(a_nx^n)' = \sum_{n=0}^{\infty}na_nx^{n-1}$$

求导后的级数与原级数有相同的收敛半径。

📐 性质3：可积性

\(S(x)\)在收敛域上可积，且可逐项积分：

$$\int_0^x S(t)dt = \int_0^x \sum_{n=0}^{\infty}a_nt^n dt = \sum_{n=0}^{\infty}\int_0^x a_nt^n dt = \sum_{n=0}^{\infty}\frac{a_n}{n+1}x^{n+1}$$

积分后的级数与原级数有相同的收敛半径。

3. 函数的幂级数展开

📐 泰勒级数

如果函数\(f(x)\)在区间\((x_0-R, x_0+R)\)上有任意阶导数，若

$$f(x) = \sum_{n=0}^{\infty}a_n(x-x_0)^n$$

则必有：

$$a_n = \frac{f^{(n)}(x_0)}{n!}, \quad n = 0,1,2,\cdots$$

即：

$$f(x) = \sum_{n=0}^{\infty}\frac{f^{(n)}(x_0)}{n!}(x-x_0)^n$$
📖 麦克劳林级数

当\(x_0 = 0\)时，泰勒级数称为麦克劳林级数：

$$f(x) = \sum_{n=0}^{\infty}\frac{f^{(n)}(0)}{n!}x^n$$

4. 常用函数的幂级数展开

(1) 几何级数：

$$\frac{1}{1-x} = 1 + x + x^2 + \cdots = \sum_{n=0}^{\infty}x^n, \quad |x| < 1$$

(2) 指数函数：

$$e^x = 1 + x + \frac{x^2}{2!} + \cdots + \frac{x^n}{n!} + \cdots = \sum_{n=0}^{\infty}\frac{x^n}{n!}, \quad x \in (-\infty, +\infty)$$

(3) 正弦函数：

$$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{(2n+1)!}, \quad x \in (-\infty, +\infty)$$

(4) 余弦函数：

$$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n}}{(2n)!}, \quad x \in (-\infty, +\infty)$$

(5) 对数函数：

$$\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots = \sum_{n=1}^{\infty}\frac{(-1)^{n-1}x^n}{n}, \quad x \in (-1, 1]$$

(6) 二项式级数：

$$(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \cdots = \sum_{n=0}^{\infty}\frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}x^n$$

当\(\alpha\)为正整数时，\(x \in \mathbb{R}\)；否则\(x \in (-1, 1)\)

第三节 傅里叶级数

📖 定义：傅里叶级数

设函数\(f(x)\)是周期为\(2\pi\)的周期函数，且在\([-\pi, \pi]\)上可积，则称

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\cos nx\, dx, \quad n = 0,1,2,\cdots$$ $$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin nx\, dx, \quad n = 1,2,\cdots$$

为\(f(x)\)的傅里叶系数，称级数

$$\frac{a_0}{2} + \sum_{n=1}^{\infty}(a_n\cos nx + b_n\sin nx)$$

为\(f(x)\)的傅里叶级数，记作：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty}(a_n\cos nx + b_n\sin nx)$$

1. 收敛定理（狄利克雷收敛定理）

📐 狄利克雷(Dirichlet)收敛定理

设\(f(x)\)在\([-\pi, \pi]\)上连续或只有有限个第一类间断点，且最多只有有限个极值点，则\(f(x)\)的傅里叶级数在\([-\pi, \pi]\)上处处收敛，且收敛于：

  • \(S(x) = f(x)\)，当\(x\)为\(f(x)\)的连续点
  • \(S(x) = \frac{f(x^-) + f(x^+)}{2}\)，当\(x\)为\(f(x)\)的间断点
  • \(S(x) = \frac{f(-\pi^+) + f(\pi^-)}{2}\)，当\(x = \pm\pi\)

2. 奇偶函数的傅里叶展开

📐 奇函数的傅里叶展开（正弦级数）

若\(f(x)\)为奇函数，则：

$$a_n = 0, \quad n = 0,1,2,\cdots$$ $$b_n = \frac{2}{\pi}\int_0^{\pi}f(x)\sin nx\, dx, \quad n = 1,2,\cdots$$
📐 偶函数的傅里叶展开（余弦级数）

若\(f(x)\)为偶函数，则：

$$a_n = \frac{2}{\pi}\int_0^{\pi}f(x)\cos nx\, dx, \quad n = 0,1,2,\cdots$$ $$b_n = 0, \quad n = 1,2,\cdots$$

3. 周期为2l的函数的傅里叶展开

📖 一般周期函数的展开

设\(f(x)\)的周期为\(2l\)，则傅里叶系数为：

$$a_n = \frac{1}{l}\int_{-l}^{l}f(x)\cos\frac{n\pi x}{l}\, dx, \quad n = 0,1,2,\cdots$$ $$b_n = \frac{1}{l}\int_{-l}^{l}f(x)\sin\frac{n\pi x}{l}\, dx, \quad n = 1,2,\cdots$$

傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty}\left(a_n\cos\frac{n\pi x}{l} + b_n\sin\frac{n\pi x}{l}\right)$$

例题：将函数\(f(x) = \pi x + x^2\) (\(-\pi < x < \pi\))展开为傅里叶级数。

解：

因为\(f(x) = \pi x + x^2\)是偶函数（注意：这里有误，\(\pi x\)是奇函数，\(x^2\)是偶函数），所以：

$$a_0 = \frac{2}{\pi}\int_0^{\pi}(\pi x + x^2)dx = \frac{2}{\pi}\left[\frac{\pi x^2}{2} + \frac{x^3}{3}\right]_0^{\pi} = \pi^2 + \frac{2\pi^2}{3}$$

$$a_n = \frac{2}{\pi}\int_0^{\pi}(\pi x + x^2)\cos nx\, dx$$

通过分部积分可得：\(a_n = \frac{2(-1)^n}{n^2}\)

$$b_n = 0 \text{（因为被积函数是奇函数）}$$

因此：

$$f(x) = \frac{\pi^2}{3} + \sum_{n=1}^{\infty}\frac{2(-1)^n}{n^2}\cos nx$$

🎓 学习小结

  • 常数项级数重点掌握：收敛判别法（比较、比值、根值、莱布尼茨）
  • 幂级数重点掌握：收敛半径、收敛域的求法，逐项求导与积分
  • 函数展开：熟记基本函数的麦克劳林展开式
  • 傅里叶级数：掌握系数计算、奇偶函数的展开
