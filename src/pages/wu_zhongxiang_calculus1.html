第一章：函数 极限 连续

一、函数的概念及常见函数

📌 函数定义

设 \(x\) 和 \(y\) 是两个变量，\(D\) 是一个给定的数集。如果对于 \(D\) 中每个数 \(x\)，按照某个对应法则 \(f\)，总有唯一确定的数值 \(y\) 和它对应，则称 \(y\) 是 \(x\) 的函数，记为：

$$y = f(x), \quad x \in D$$

其中：

  • \(x\) 称为自变量
  • \(y\) 称为因变量
  • \(D\) 称为函数的定义域，记作 \(D_f = D\)
  • 函数值的全体构成的集合称为函数的值域，记作 \(R_f\) 或 \(f(D)\)
⚠️ 注意：函数有两个基本要素：定义域和对应法则。当两个函数的定义域和对应法则完全相同时，它们就是同一函数。

1. 符号函数（Sign Function）

定义： $$y = \text{sgn}\,x = \begin{cases} -1, & x < 0 \\ 0, & x = 0 \\ 1, & x > 0 \end{cases}$$

2. 取整函数（Floor Function）

定义：

设 \(x\) 为任意实数，不超过 \(x\) 的最大整数称为 \(x\) 的整数部分，记为 \([x]\) 或 \(\lfloor x \rfloor\)，称为取整函数。

基本性质： $$x - 1 < [x] \leq x < [x] + 1, \quad x \in \mathbb{R}$$

3. 复合函数（Composite Function）

📌 复合函数定义

设 \(y = f(u)\) 的定义域为 \(D_f\)，\(u = g(x)\) 的定义域为 \(D_g\)，值域为 \(R_g\)，如果 \(D_g \cap D_f \neq \emptyset\)，则由 \(y = f[g(x)]\) 确定的函数称为复合函数。

其定义域为：$$\{x \mid x \in D_g, g(x) \in D_f\}$$

💡 重要提示：不是任何两个函数都可以复合！例如，\(y = f(u) = \ln u\) （\(u > 0\)）与 \(u = g(x) = \sin x - 1\) （\(R_g = [-2, 0]\)）无法复合，因为 \(D_g \cap R_g = \emptyset\)。

4. 反函数（Inverse Function）

📌 反函数定义

设函数 \(y = f(x)\) 的定义域为 \(D\)，值域为 \(R_y\)。如果对于 \(R_y\) 中每个 \(y\)，在 \(D\) 中都有唯一确定的 \(x\) 使得 \(y = f(x)\)，则由此定义的函数 \(x = \varphi(y)\) 称为 \(y = f(x)\) 的反函数。

📝 反函数性质：
  • 不是每个函数都有反函数（例如 \(y = x^2\) 没有反函数）
  • 单调函数一定有反函数，但反之不然
  • 有时可用 \(y = f^{-1}(x)\) 表示反函数
  • 函数 \(y = f(x)\) 与其反函数 \(x = f^{-1}(y)\) 的图形关于直线 \(y = x\) 对称
  • \(f[f^{-1}(x)] = x\)，\(f^{-1}[f(x)] = x\)

5. 基本初等函数

幂函数、指数函数、对数函数、三角函数及反三角函数统称为基本初等函数。

📊 幂函数：\(y = x^\alpha\) （\(\alpha\) 为实数）

常见幂函数：

  • \(y = x\), \(y = x^2\), \(y = x^3\)
  • \(y = \sqrt{x}\), \(y = \sqrt[3]{x}\)
  • \(y = \frac{1}{x}\)

📈 指数函数：\(y = a^x\) （\(a > 0, a \neq 1\)）

性质：

  • 定义域：\((-\infty, +\infty)\)
  • 值域：\((0, +\infty)\)
  • 当 \(a > 1\) 时，\(y = a^x\) 单调递增
  • 当 \(0 < a < 1\) 时，\(y = a^x\) 单调递减
  • 重要：$$\lim_{x \to -\infty} e^x = 0, \quad \lim_{x \to +\infty} e^x = +\infty$$

📉 对数函数：\(y = \log_a x\) （\(a > 0, a \neq 1\)）

性质：

  • 定义域：\((0, +\infty)\)
  • 值域：\((-\infty, +\infty)\)
  • 当 \(a > 1\) 时，\(y = \log_a x\) 单调递增
  • 当 \(0 < a < 1\) 时，\(y = \log_a x\) 单调递减
  • 重要：$$\lim_{x \to 0^+} \ln x = -\infty, \quad \lim_{x \to +\infty} \ln x = +\infty$$

二、函数的性质

1. 单调性

设 \(y = f(x)\) 在某区间 \(I\) 上有定义，如果对于区间 \(I\) 内的任意两点 \(x_1 < x_2\)，恒有：

  • \(f(x_1) < f(x_2)\) 或 \(f(x_1) > f(x_2)\)，则称 \(y = f(x)\) 在该区间内单调递增（或单调递减）

2. 奇偶性

设 \(y = f(x)\) 的定义域 \(D\) 关于原点对称，若对任意 \(x \in D\)，如果恒有：

  • \(f(-x) = f(x)\)，则称 \(f(x)\) 为 \(D\) 上的偶函数
  • \(f(-x) = -f(x)\)，则称 \(f(x)\) 为 \(D\) 上的奇函数
📝 奇偶函数性质：
  • 奇函数的图形关于原点对称
  • 偶函数的图形关于 \(y\) 轴对称
  • 两个奇（偶）函数之和仍为奇（偶）函数
  • 两个奇（偶）函数之积为偶函数
  • 奇函数 × 偶函数 = 奇函数

3. 周期性

设 \(y = f(x)\) 的定义域为 \(D\)，如果存在实数 \(T > 0\)，使得对于任意 \(x \in D\)，恒有 \(f(x + T) = f(x)\)，则称 \(f(x)\) 为以 \(T\) 为周期的周期函数。

💡 周期函数示例：
  • \(\sin x\) 和 \(\cos x\) 以 \(2\pi\) 为周期
  • \(\sin 2x\), \(|\sin x|\) 以 \(\pi\) 为周期
  • \(\tan x\), \(\cot x\) 以 \(\pi\) 为周期
  • 若 \(f(x)\) 以 \(T\) 为周期，则 \(f(ax + b)\) 以 \(\frac{T}{|a|}\) 为周期

4. 有界性

设 \(y = f(x)\) 在集合 \(X\) 上有定义。若存在 \(M > 0\)，使得对任意 \(x \in X\)，恒有：

$$|f(x)| \leq M$$

则称 \(f(x)\) 在 \(X\) 上为有界函数。否则称 \(f(x)\) 在 \(X\) 上为无界函数。

📝 常见有界函数：
  • \(|\sin x| \leq 1\), \(|\cos x| \leq 1\)
  • \(|\arcsin x| \leq \frac{\pi}{2}\), \(|\arctan x| < \frac{\pi}{2}\)
  • \(|\arccos x| \leq \pi\)

一、极限的概念

1. 数列的极限

📌 数列极限定义（\(\varepsilon-N\) 语言）

如果对于任意给定的 \(\varepsilon > 0\)，总存在正整数 \(N\)，当 \(n > N\) 时，恒有：

$$|x_n - a| < \varepsilon$$

成立，则称常数 \(a\) 为数列 \(\{x_n\}\) 当 \(n \to \infty\) 时的极限，记为：

$$\lim_{n \to \infty} x_n = a$$

💡 几何意义：\(\varepsilon\) 刻画 \(x_n\) 与 \(a\) 的接近程度，\(N\) 是用来刻画极限过程的。对于任何邻域 \((a - \varepsilon, a + \varepsilon)\)，一定存在 \(N\)，当 \(n > N\) 时，第 \(N\) 项以后的所有项都落在开区间 \((a - \varepsilon, a + \varepsilon)\) 内，而有限项（最多有 \(N\) 项）在这区间之外。

2. 函数的极限

📌 函数极限定义（\(x \to x_0\) 时）

设函数 \(f(x)\) 在点 \(x_0\) 的某去心邻域内有定义。如果对于任意给定的 \(\varepsilon > 0\)，总存在 \(\delta > 0\)，当 \(0 < |x - x_0| < \delta\) 时，恒有：

$$|f(x) - A| < \varepsilon$$

成立，则称常数 \(A\) 为函数 \(f(x)\) 当 \(x \to x_0\) 时的极限，记为：

$$\lim_{x \to x_0} f(x) = A$$

📌 函数极限定义（\(x \to \infty\) 时）

若对于任意给定的 \(\varepsilon > 0\)，总存在 \(X > 0\)，当 \(|x| > X\) 时，恒有：

$$|f(x) - A| < \varepsilon$$

成立，则称常数 \(A\) 为函数 \(f(x)\) 当 \(x \to \infty\) 时的极限，记为：

$$\lim_{x \to \infty} f(x) = A$$

3. 左极限与右极限

左极限： $$\lim_{x \to x_0^-} f(x) = f(x_0 - 0) = A$$ 右极限： $$\lim_{x \to x_0^+} f(x) = f(x_0 + 0) = A$$

⭐ 重要定理

极限 \(\lim_{x \to x_0} f(x)\) 存在的充要条件是：极限 \(\lim_{x \to x_0^-} f(x)\) 和 \(\lim_{x \to x_0^+} f(x)\) 都存在且相等。

二、极限的性质

1. 唯一性

如果数列 \(\{x_n\}\) 收敛，则它的极限必定唯一。

2. 保号性

设 \(\lim_{n \to \infty} x_n = a\)。

  • 如果 \(a > 0\)（或 \(a < 0\)），则存在 \(N > 0\)，当 \(n > N\) 时，\(x_n > 0\)（或 \(x_n < 0\)）
  • 如果存在 \(N > 0\)，当 \(n > N\) 时，\(x_n \geq 0\)（或 \(x_n \leq 0\)），则 \(a \geq 0\)（或 \(a \leq 0\)）
⚠️ 注意：结论 (1) 中是 ">" 或 "<"；结论 (2) 中是 "\(\geq\)" 或 "\(\leq\)"。

三、极限的存在准则

1. 夹逼准则

⭐ 夹逼准则（数列）

若存在 \(N\)，当 \(n > N\) 时，\(x_n \leq y_n \leq z_n\)，且 \(\lim_{n \to \infty} x_n = \lim_{n \to \infty} z_n = a\)，则 \(\lim_{n \to \infty} y_n = a\)。

2. 单调有界准则

⭐ 单调有界准则

单调有界数列必有极限，即单调增（减）且有上（下）界的数列必有极限。

四、无穷小量

1. 无穷小量的概念

若函数 \(f(x)\) 当 \(x \to x_0\) （或 \(x \to \infty\)）时趋向于零，则称 \(f(x)\) 为 \(x \to x_0\) （或 \(x \to \infty\)）时的无穷小量。

2. 无穷小的比较

设 \(\lim \alpha(x) = 0\)，\(\lim \beta(x) = 0\)，且 \(\beta(x) \neq 0\)。

  • 高阶：如果 \(\lim \frac{\alpha(x)}{\beta(x)} = 0\)，记作 \(\alpha(x) = o(\beta(x))\)
  • 低阶：如果 \(\lim \frac{\alpha(x)}{\beta(x)} = \infty\)
  • 同阶：如果 \(\lim \frac{\alpha(x)}{\beta(x)} = C \neq 0\)
  • 等价：如果 \(\lim \frac{\alpha(x)}{\beta(x)} = 1\)，记作 \(\alpha(x) \sim \beta(x)\)
  • k 阶：如果 \(\lim \frac{\alpha(x)}{[\beta(x)]^k} = C \neq 0\)，称 \(\alpha(x)\) 是 \(\beta(x)\) 的 \(k\) 阶无穷小

3. 常用的等价无穷小（当 \(x \to 0\) 时）

⭐ 必记等价无穷小： $$\begin{align} x &\sim \sin x \sim \tan x \sim \arcsin x \sim \arctan x \sim \ln(1+x) \sim e^x - 1 \\ (1+x)^\alpha - 1 &\sim \alpha x \quad (\alpha \neq 0) \\ 1 - \cos x &\sim \frac{1}{2}x^2 \\ a^x - 1 &\sim x \ln a \\ x - \sin x &\sim \frac{1}{6}x^3 \\ \tan x - x &\sim \frac{1}{3}x^3 \\ x - \ln(1+x) &\sim \frac{1}{2}x^2 \\ \arcsin x - x &\sim \frac{1}{6}x^3 \\ x - \arctan x &\sim \frac{1}{3}x^3 \end{align}$$

五、无穷大量

若函数 \(f(x)\) 当 \(x \to x_0\) （或 \(x \to \infty\)）时趋向于无穷，则称 \(f(x)\) 为 \(x \to x_0\) （或 \(x \to \infty\)）时的无穷大量。

常用的一些重要结论（当 \(x \to +\infty\) 时）

$$\ln^\beta x \ll x^\alpha \ll a^x$$

其中 \(\alpha > 0\), \(\beta > 0\), \(a > 1\)。

口诀："对数 < 幂函数 < 指数函数"

求极限的方法（8种）

方法 1：利用基本极限求极限

⭐ 常用的基本极限： $$\begin{align} &\lim_{x \to 0} \frac{\sin x}{x} = 1 \\ &\lim_{x \to 0} (1+x)^{\frac{1}{x}} = e \\ &\lim_{n \to \infty} \left(1 + \frac{1}{x}\right)^x = e \\ &\lim_{x \to 0} \frac{a^x - 1}{x} = \ln a \\ &\lim_{n \to \infty} \sqrt[n]{n} = 1 \\ &\lim_{n \to \infty} \sqrt[n]{a} = 1 \quad (a > 0) \end{align}$$

方法 2：利用等价无穷小代换求极限

📝 代换原则：
  • 乘除关系可以代换
  • 加减关系在一定条件下可以代换

方法 3：利用有理运算法则求极限

若 \(\lim f(x) = A\)，\(\lim g(x) = B\)，则：

$$\begin{align} \lim [f(x) \pm g(x)] &= \lim f(x) \pm \lim g(x) = A \pm B \\ \lim [f(x) \cdot g(x)] &= \lim f(x) \cdot \lim g(x) = AB \\ \lim \frac{f(x)}{g(x)} &= \frac{\lim f(x)}{\lim g(x)} = \frac{A}{B} \quad (B \neq 0) \end{align}$$

方法 4：利用洛必达法则求极限

⭐ 洛必达法则

若：

  1. \(\lim f(x) = \lim g(x) = 0\) （或 \(\infty\)）
  2. \(f(x)\) 和 \(g(x)\) 在 \(x_0\) 的某去心邻域内可导，且 \(g'(x) \neq 0\)
  3. \(\lim \frac{f'(x)}{g'(x)}\) 存在（或为 \(\infty\)）

则：$$\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}$$

⚠️ 洛必达法则适用类型：

洛必达法则可用来求七种类型的未定式极限，即 \(\frac{0}{0}\)，\(\frac{\infty}{\infty}\)，\(\infty - \infty\)，\(0 \cdot \infty\)，\(1^\infty\)，\(\infty^0\)，\(0^0\)，其中前两种 \(\frac{0}{0}\)，\(\frac{\infty}{\infty}\) 可以直接用洛必达法则，后五种可化为前两种。

📝 例题：求 \(\lim_{x \to 0} \frac{\cos x - e^{-\frac{x^2}{2}}}{x^4}\)

解法一：由泰勒公式知

$$\begin{align} \cos x &= 1 - \frac{x^2}{2!} + \frac{x^4}{4!} + o(x^4) \\ e^{-\frac{x^2}{2}} &= 1 - \frac{x^2}{2} + \frac{1}{2!}\left(-\frac{x^2}{2}\right)^2 + o(x^4) \end{align}$$

则原式 = \(\lim_{x \to 0} \frac{-\frac{1}{12}x^4 + o(x^4)}{x^4} = -\frac{1}{12}\)

一、连续性的概念

📌 连续性定义

设 \(y = f(x)\) 在点 \(x_0\) 的某邻域内有定义，若：

$$\lim_{\Delta x \to 0} \Delta y = \lim_{\Delta x \to 0} [f(x_0 + \Delta x) - f(x_0)] = 0$$

则称 \(y = f(x)\) 在点 \(x_0\) 连续。

💡 连续的等价定义：

\(y = f(x)\) 在点 \(x_0\) 连续的充要条件是：

  • \(f(x)\) 在 \(x_0\) 有定义
  • \(\lim_{x \to x_0} f(x)\) 存在
  • \(\lim_{x \to x_0} f(x) = f(x_0)\)

左连续与右连续

左连续： $$\lim_{x \to x_0^-} f(x) = f(x_0 - 0) = f(x_0)$$ 右连续： $$\lim_{x \to x_0^+} f(x) = f(x_0 + 0) = f(x_0)$$

⭐ 重要定理

函数 \(f(x)\) 在点 \(x_0\) 处连续的充要条件是 \(f(x)\) 在点 \(x_0\) 处既左连续又右连续。

二、间断点及其分类

📌 间断点定义

若 \(f(x)\) 在 \(x_0\) 的某邻域内有定义，若在 \(x_0\) 处不连续，则称 \(x_0\) 为 \(f(x)\) 的间断点。

间断点的分类

第一类间断点：左、右极限都存在的间断点
  • 可去间断点：左、右极限都存在且相等的间断点
  • 跳跃间断点：左、右极限都存在但不相等的间断点
第二类间断点：左、右极限至少有一个不存在的间断点
  • 无穷间断点：\(\lim_{x \to x_0^-} f(x) = \infty\) 或 \(\lim_{x \to x_0^+} f(x) = \infty\)
  • 振荡间断点：例如 \(y = \sin \frac{1}{x}\) 在 \(x = 0\) 处没定义且极限都不存在（在 \(x \to 0\) 时其函数值在 \(-1\) 与 \(1\) 之间多次振荡）

三、连续性的运算与性质

若 \(f(x)\) 和 \(g(x)\) 在点 \(x_0\) 连续，则 \(f(x) \pm g(x)\)，\(f(x) \cdot g(x)\)，\(\frac{f(x)}{g(x)}\) （\(g(x_0) \neq 0\)）在点 \(x_0\) 也连续。

设函数 \(u = \varphi(x)\) 在点 \(x = x_0\) 连续，且 \(\varphi(x_0) = u_0\)，而函数 \(y = f(u)\) 在点 \(u = u_0\) 连续，则复合函数 \(y = f[\varphi(x)]\) 在点 \(x = x_0\) 连续。

四、闭区间上连续函数的性质

⭐ 最大值最小值定理

设 \(f(x)\) 在闭区间 \([a, b]\) 上连续，则 \(f(x)\) 在 \([a, b]\) 上必有最大值与最小值。

⭐ 有界性定理

设 \(f(x)\) 在闭区间 \([a, b]\) 上连续，则 \(f(x)\) 在 \([a, b]\) 上必有界。

⭐ 介值定理

设 \(f(x)\) 在 \([a, b]\) 上连续，且 \(f(a) \neq f(b)\)，则对于任意介于 \(f(a)\) 与 \(f(b)\) 之间的数 \(C\)，至少存在一点 \(\xi \in (a, b)\)，使 \(f(\xi) = C\)。

⭐ 零点定理

设 \(f(x)\) 在 \([a, b]\) 上连续，且 \(f(a) \cdot f(b) < 0\)，则至少存在一点 \(\xi \in (a, b)\)，使 \(f(\xi) = 0\)。

💡 应用：零点定理的一个重要应用就是证明方程的根的存在性。
