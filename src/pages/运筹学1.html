📊 线性规划与单纯形法

运筹学教程 · 第一章 · 交互式学习

🌟 什么是线性规划？

线性规划是运筹学中的一个重要分支，用于在一定约束条件下，寻找使目标函数达到最优的决策方案。

🏭 典型应用场景

  • ✅ 生产计划安排
  • ✅ 资源分配优化
  • ✅ 运输问题
  • ✅ 投资组合
  • ✅ 人员调度

📋 线性规划的三要素

  1. 决策变量：需要决定的未知量（如生产多少件产品）
  2. 目标函数：要优化的指标（如利润最大、成本最小）
  3. 约束条件：决策变量需要满足的限制（如资源限制）
💡 提示：线性规划要求目标函数和约束条件都是线性的，即变量的一次函数。

📐 线性规划的数学模型

标准形式

max z = c₁x₁ + c₂x₂ + ... + cₙxₙ s.t. a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁ a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂ ... aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ = bₘ x₁, x₂, ..., xₙ ≥ 0

🏢 例子：美佳公司生产问题

问题描述：公司生产家电I和II，每件占用设备时间和利润如下：

项目 家电 I 家电 II 每天可用能力
设备 A (h) 0 5 15
设备 B (h) 6 2 24
调试工序 (h) 1 1 5
利润 (元) 2 1 -

数学模型：

max z = 2x₁ + x₂ s.t. 5x₂ ≤ 15 6x₁ + 2x₂ ≤ 24 x₁ + x₂ ≤ 5 x₁, x₂ ≥ 0
✅ 建模技巧：先确定决策变量，再根据题意写出目标函数和约束条件。

📈 图解法（仅适用于2个变量）

对于只有两个决策变量的线性规划问题，可以用图解法直观地找到最优解。

🎮 交互式演示

📝 步骤：
  1. 画出所有约束条件的直线
  2. 找出可行域（所有约束的交集）
  3. 画出目标函数的等值线
  4. 沿着优化方向移动等值线，找到最优点

🎯 可能的求解结局

  • 唯一最优解：目标函数在可行域的某个顶点达到最优
  • 无穷多最优解：目标函数等值线与可行域某条边重合
  • 无界解：可行域无界，目标函数可无限增大
  • 无可行解：约束条件相互矛盾，不存在可行域

🔢 单纯形法

单纯形法是求解线性规划问题的通用算法，适用于任意多个变量的情况。

🎯 核心思想

从可行域的一个顶点出发，沿着可行域的边移动到相邻的顶点， 使目标函数值不断增大，直到找到最优解。

📋 计算步骤

  1. 化为标准形式：添加松弛变量，使不等式变为等式
  2. 找初始基可行解：令非基变量=0，求出基变量的值
  3. 最优性检验：计算检验数 σⱼ = cⱼ - zⱼ，若都≤0则最优
  4. 确定换入变量：选择 σⱼ > 0 中最大的对应变量
  5. 确定换出变量：用最小比值法确定主元素
  6. 迭代：进行行变换，得到新的基可行解
  7. 重复步骤3-6：直到找到最优解

📊 单纯形表

Cʙ 基 b x₁ x₂ x₃ x₄ x₅
0 x₃ 15 0 5 1 0 0
0 x₄ 24 [6] 2 0 1 0
0 x₅ 5 1 1 0 0 1
σⱼ 0 2 1 0 0 0
💡 说明：
  • 蓝色标记：最大检验数，对应换入变量 x₁
  • 红色标记：主元素（最小比值法确定）
  • 主元素所在行的基变量 x₄ 是换出变量
✅ 最优解：x₁ = 7/2, x₂ = 3/2, 最大利润 z = 17/2 = 8.5 元

💡 实际应用实例

🏭 实例1：混合配料问题

问题：糖果厂用原料A、B、C加工三种糖果甲、乙、丙。已知：

  • 各种糖果中原料含量要求
  • 原料成本和月限制用量
  • 加工费和售价

目标：确定每月生产各种糖果的数量，使利润最大。

🚌 实例2：公交公司人员调度

问题：海河市公交公司需安排司机，每天各时段所需人数不同，每人连续工作6小时。

目标：

  • 确定最少需要配备多少人员
  • 如何安排倒班和休息日才能公平合理

🏦 实例3：投资规划

问题：宏银公司需为建设项目提供4年贷款，可通过购买不同期限债券或银行存款筹资。

目标：在满足每年贷款需求的前提下，使筹资总额最小。

关键点：需要考虑不同投资期限和收益率的组合优化。

🎯 实践练习

尝试建立以下问题的数学模型：

问题：某工厂生产甲、乙两种产品，每件产品需要A、B两种原料：

  • 甲产品：需要A原料2kg，B原料3kg，利润5元/件
  • 乙产品：需要A原料4kg，B原料2kg，利润6元/件
  • 每天A原料供应16kg，B原料供应12kg

求：生产多少甲、乙产品使利润最大？

数学模型：
设 x₁ = 甲产品数量，x₂ = 乙产品数量 max z = 5x₁ + 6x₂ s.t. 2x₁ + 4x₂ ≤ 16 (A原料限制) 3x₁ + 2x₂ ≤ 12 (B原料限制) x₁, x₂ ≥ 0
