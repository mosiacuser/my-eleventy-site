1️⃣ 事件的关系与运算

基本概念

相容：若 $AB \neq \emptyset$，则称事件 $A$ 和 $B$ 相容。

互斥：若 $AB = \emptyset$，则称事件 $A$ 与 $B$ 互不相容，也叫互斥。

💡 注意：如果一些事件中任意两个事件都互斥，则称这些事件是两两互斥的，或简称互斥的。

$$P(A \cup B) = P(A) + P(B) \quad \text{（加法公式）}$$

称"事件 $A$ 与 $B$ 至少有一个发生"的事件为事件 $A$ 与 $B$ 的和事件（或并事件），记为 $A \cup B$。

推广：称"有限个（或可列个）事件 $A_1, A_2, \ldots, A_n, \ldots$ 至少有一个发生"的事件为这些事件的和事件，记为：

$$\bigcup_{i=1}^{n} A_i \quad \text{或} \quad \bigcup_{i=1}^{\infty} A_i$$

差：称"事件 $A$ 发生而事件 $B$ 不发生"的事件为事件 $A$ 与 $B$ 的差事件，记为 $A - B$。

逆（对立）：称"事件 $A$ 不发生"的事件为事件 $A$ 的逆事件或对立事件，记为 $\overline{A}$。

⚠️ 重要区别：

  • 对立事件一定是互不相容事件
  • 但互不相容事件不一定是对立事件

例如：$A$ 与 $B$，$B$ 与 $C$，$A$ 与 $C$ 均为互斥事件，但 $B \cup C$ 才是 $A$ 的对立事件。

重要关系：

$$A - B = A - AB = A\overline{B}$$ $$\overline{B} = \overline{A} \Leftrightarrow AB = \emptyset \text{ 且 } A \cup B = \Omega$$

如果 $\bigcup_{i=1}^{n} A_i = \Omega$（或 $\bigcup_{i=1}^{\infty} A_i = \Omega$），且 $A_i A_j = \emptyset$（对一切 $i \neq j$；$i, j = 1, 2, \ldots, n, \ldots$），称有限个（或可列个）事件 $A_1, A_2, \ldots, A_n, \ldots$ 构成一个完备事件组。

💡 注：完备事件组也称为对样本空间的一种划分，也叫全集分解。

3️⃣ 运算法则

若 $A \subset B$，则：

$$A \cup B = B, \quad A \cap B = A$$
$$A \cup B = B \cup A$$ $$A \cap B = B \cap A$$
$$(A \cup B) \cup C = A \cup (B \cup C)$$ $$(A \cap B) \cap C = A \cap (B \cap C)$$
$$A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$$ $$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$$ $$A \cap (B - C) = (A \cap B) - (A \cap C)$$
$$\overline{A \cup B} = \overline{A} \cap \overline{B}$$ $$\overline{A \cap B} = \overline{A} \cup \overline{B}$$

🎯 口诀：长杠变短杠；开口换方向

例如：

$$\overline{A \cap B} = \overline{A} \cup \overline{B} \Rightarrow \overline{A \cup B}$$

💡 注意事项：

  1. 事件运算顺序约定为：先进行逆运算，然后进行交运算，最后进行并或差运算。
  2. 事件的关系、运算与集合的关系、运算相当，且具有相同的运算法则，所以可以对比着理解记忆，并要学会用集合关系去考虑事件关系。

4️⃣ 概率的定义

（1）描述性定义

通常将随机事件 $A$ 发生的可能性大小的度量（非负值）称为事件 $A$ 发生的概率，记为 $P(A)$。

📌 例子：抛硬币 1000 次，正面朝上的次数为 527，那么 $\frac{527}{1000} = 0.527$ 称为这 1000 次试验中，硬币正面朝上的频率。

（2）统计性定义

在相同条件下做重复试验，事件 $A$ 出现的次数 $k$ 和总的试验次数 $n$ 之比 $\frac{k}{n}$ 称为事件 $A$ 在这 $n$ 次试验中出现的频率。

当试验次数 $n$ 充分大时，频率将"稳定"于某常数 $p$。$n$ 越大，频率偏离这个常数 $p$ 的可能性越小。这个常数 $p$ 就称为事件 $A$ 的概率。

🤔 思考：

抛硬币正面朝上的频率是 0.527，但它是硬币正面朝上的概率吗？

  • 对于均匀的硬币，正面朝上的概率应该是 $\frac{1}{2}$ — 这是客观规律
  • 理想情况下，均匀硬币是真正的均匀，此时就说它的概率是客观上的 $\frac{1}{2}$
  • 频率只能是逐渐地靠近 $\frac{1}{2}$
  • 但在硬币不均匀的情况下，不清楚抛出硬币正面朝上的概率，不能直接说是 $\frac{1}{2}$

⚠️ 重要理解：

  1. 概率的统计性定义实质上是说，用频率 $\frac{k}{n}$ 作为事件 $A$ 的概率 $P(A)$ 的估计。其直观理解为某事件出现的可能性大小，可由其在多次重复试验中出现的频率去刻画。
  2. 从上述可以看出，频率只是概率的估计，而非概率本身。统计性定义的重要性主要基于以下两点：
    • ① 它提供了估计概率的方法（例如：在一批产品中抽取样品，来估计该批产品的合格率）
    • ② 它提供了一种检验某结论是否正确的准则（例如：你说某批产品的合格率是 95%，我们做试验抽取样品进行计算，得出的结果是合格率为 20%，远远低于你所说的 95%，于是拒绝你的结论）
  3. 统计性定义它没有理论上的价值，不能精确定义概率，但是从实践上来讲，它有时候是很有用的。

（3）公理化定义

设随机试验的样本空间为 $\Omega$，如果对每一个事件 $A$ 都有一个确定的实数 $P(A)$，且事件函数 $P(\cdot)$ 满足：

① 非负性：$P(A) \geq 0$

② 规范性：$P(\Omega) = 1$ （归一性）

③ 可列可加性：对任意可列个两两互不相容事件 $A_1, A_2, \ldots, A_n, \ldots$（即 $A_i A_j = \emptyset$，$i \neq j$；$i, j = 1, 2, \ldots$），有

$$P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$$

事件和的概率等于概率的和

则称 $P(\cdot)$ 为概率，$P(A)$ 为事件 $A$ 的概率。

💡 注：

  1. 数学上所说的"公理"，就是一些不加证明而承认的前提。上述公理化定义只是界定了概率这个概念所必须满足的一些一般性质，它不解决具体场合下的概率计算。
  2. 概率 $P$ 是事件的函数。
  3. 虽然公理化定义不解决具体场合下的概率计算，但是我们却常常用它来判断某事件函数 $P(\cdot)$ 是不是概率，这种题型在考研试题中也是经常遇到的。

5️⃣ 古典概型与几何概型

古典概型

（1）概念

如果随机试验的样本空间满足：

  1. 只有有限个样本点（基本事件）
    → 如掷骰子，$\Omega = \{\omega_1, \omega_2, \ldots, \omega_6\}$。一旦出现无穷个，就不是古典概型
  2. 每个样本点（基本事件）发生的可能性都一样
    ✓ 具有等可能性

则称随机试验的概率模型为古典概型。

📐 概率计算公式：

如果古典概型的基本事件总数为 $n$，事件 $A$ 包含 $k$ 个基本事件（也叫作有利于 $A$ 的基本事件有 $k$ 个），则：

$$P(A) = \frac{k}{n} = \frac{\text{事件 } A \text{ 所含基本事件的个数}}{\text{基本事件总数}}$$

例：掷骰子，事件 $A = \{\text{掷出偶数点}\}$，则 $P(A) = \frac{3}{6} = \frac{1}{2}$

由上式计算得出的概率称为 $A$ 的古典概率。

💡 注：计算的关键是基本事件、样本空间的选定以及基本事件数的计算。

6️⃣ 计数方法

计数方法常用的有三种：

基本事件数不多时常用这种方法。

① 加法原理

完成一件事有 $n$ 类办法，第一类办法中有 $m_1$ 种方法，第二类办法中有 $m_2$ 种方法，…，第 $n$ 类办法中有 $m_n$ 种方法，则完成此事共有：

$$\sum_{i=1}^{n} m_i \text{ 种方法}$$

② 乘法原理

完成一件事有 $n$ 个步骤，第一步有 $m_1$ 种方法，第二步有 $m_2$ 种方法，…，第 $n$ 步有 $m_n$ 种方法，则完成此事共有：

$$\prod_{i=1}^{n} m_i \text{ 种方法}$$

📌 例子：从 A 地到 B 地的方法

方式1（加法原理）：

  • 飞机：C919, C929, A330（3种）
  • 火车：复兴号, 和谐号（2种）
  • 总计：$m_1 + m_2 = 3 + 2 = 5$ 种交通方式

方式2（乘法原理）：

需要2个步骤：

  • 第1步：坐飞机（3种选择）
  • 第2步：坐火车（2种选择）
  • 总计：$m_1 \cdot m_2 = 3 \times 2 = 6$ 种交通方式

排列数公式：

$$A_n^m = \frac{n!}{(n-m)!} = n(n-1)(n-2)\cdots(n-m+1)$$

组合数公式：

$$C_n^m = \binom{n}{m} = \frac{n!}{m!(n-m)!} = \frac{A_n^m}{m!}$$

🎯 记忆技巧：

  • 排列：有顺序要求（Arrangement）
  • 组合：无顺序要求（Combination）

🎯 互动测验

问题1：互斥事件和对立事件的区别是什么？

答案：

  • 互斥事件：$AB = \emptyset$（两个事件不能同时发生）
  • 对立事件：$AB = \emptyset$ 且 $A \cup B = \Omega$（两个事件不能同时发生，但必有一个发生）
  • 结论：对立事件一定是互斥事件，但互斥事件不一定是对立事件

问题2：德·摩根律的口诀是什么？

答案：长杠变短杠；开口换方向

即：$\overline{A \cup B} = \overline{A} \cap \overline{B}$ 和 $\overline{A \cap B} = \overline{A} \cup \overline{B}$
