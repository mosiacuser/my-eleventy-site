第一部分：求解物理应用题（仅数学一、数学二）

三问解题法

求解物理应用题（仅数学一、数学二）
（O(Ⅱ)任目标))

\(W = \int_a^b F(x)dx\)

\(W = \rho g\int_a^b xA(x)dx\)
(D₁(常规操作))

\(\int_d^b \frac{Gm\mu}{(a-x)^2}dx\)
(D₁(常规操作))
↓
变速直线运动的位移
\(s = \int_{t_1}^{t_2} v(t)dt\)
(D₁(常规操作))
静水压力
\(P = \rho g\int_a^b x[f(x)-h(x)]dx\)
(D₁(常规操作))
用微元法自行建立表达式
(D₁(常规操作)+D₂₂(转换等价表述))

设方向沿x轴正向的力函数为 \(F(x)\) (\(a \leq x \leq b\))，则物体沿x轴从A点a移动到点b时，变力 \(F(x)\) 所做的功（见图）为：

$$W = \int_a^b F(x)dx$$

功的元素：\(dW = F(x)dx\)

例12.1

如图所示，并联a米，每米绳子的重量是5 N，挂斗重400 N，污泥重1 500 N，将挂斗从井底提到井口所做的功为59 250 J，则 \(a =\) _______。

【解】应填30。

$$W = \int_0^a [400 + 1500 + 5(a-x)]dx = 59\ 250\ (J)$$

解得 \(a = 30\) (因为a为井深，大于0，所以负值舍去)。

设一物体在 t 时刻的运动速度为 \(v(t)\)，则从 \(t_1\) 时刻到 \(t_2\) 时刻所经过的路程为：

$$s = \int_{t_1}^{t_2} v(t)dt$$
例12.2

设某物体运动的路程 s 和加速度 a 与时间 t 的关系分别为 \(s = s(t)\)，\(a = a(t)\)，已知该物体从 \(t=0\) 时刻至 \(t=1\) 时刻运动的路程是2，在 \(t=1\) 时刻的速度为2，则 \(\int_0^1 ta(t)dt =\) ( )。

(A)0     (B)1     (C)2     (D)3

【解】应选(A)。

因路程 s 与速度 v 关于时间 t 的关系满足 \(s'(t) = v(t)\)，速度 v 与加速度 a 关于时间 t 的关系满足 \(v'(t) = a(t)\)，则

$$\int_0^1 ta(t)dt = \int_0^1 t d[v(t)] = tv(t)\Big|_0^1 - \int_0^1 v(t)dt$$

$$= v(1) - [s(1) - s(0)] = 2 - 2 = 0$$

如图所示，将容器中的水全部抽出所做的功为：

$$W = \rho g\int_a^b xA(x)dx$$

其中 ρ 为水的密度，g 为重力加速度。

功的元素 \(dW = \rho gxA(x)dx\) 为位于x处厚度为dx，水平截面面积为 \(A(x)\) 的一层水被抽出(微程为x)所做的功。

求解这类问题的关键是确定x处的水平截面面积 \(A(x)\)，其余的量都是固定的。
例12.3

半径为a的半球形水池蓄满了水，水的比重为1，现将水抽干，至少做功 \(\frac{\pi}{2}\)，则 \(a =\) _______。

【解】应填 \(2^{\frac{1}{4}}\)。

如图所示，把水看作是一层一层抽出来的。

任取一个与池面距离为h的小薄层，厚度为dh，它的重量为 \(\pi(a^2 - h^2)dh\)，把这层水(微元)抽到池面所做的功是

$$dW = \pi(a^2 - h^2)hdh$$

所以抽干水所做的功至少为

$$W = \int_0^a \pi(a^2 - h^2)hdh = \pi\left(\frac{a^2}{2}h^2 - \frac{h^4}{4}\right)\Big|_0^a = \frac{\pi}{4}a^4 = \frac{\pi}{2}$$

故 \(a = 2^{\frac{1}{4}}\)。

垂直浸没在水中的平板ABCD (见图) 的一侧受到的水压力为：

$$P = \rho g\int_a^b x[f(x) - h(x)]dx$$

其中 ρ 为水的密度，g 为重力加速度。

压力元素：$$dP = \rho gx[f(x) - h(x)]dx$$

即图中矩形条所受到的压力，x 表示水深，\(f(x) - h(x)\) 是矩形条的宽度，dx 是矩形条的高度。

静水压力问题的特点：压强随水的深度的改变而改变，求解这类问题的关键是确定水深x处的平板的宽度 \(f(x) - h(x)\)。
例12.4

如图所示，一闸门的上部是一个宽为2 m，高为H m的矩形，下部由 \(y = x^2\) 与 \(y = 1\) 围成。当闸门上边缘与水面在一个平面时，其上部所受水压力与下部所受水压力之比为 \(\frac{5}{4}\)，求上部的高度H。

【解】闸门上部所受水压力为

$$P_1 = \int_1^{H+1} \rho g(H+1-y) \cdot 2dy = \rho gH^2$$

其中 ρ 为水的密度，g 为重力加速度。

闸门下部所受水压力为：

$$P_2 = \int_0^1 \rho g(H+1-y) \cdot 2\sqrt{y}dy = \frac{4}{3}\rho g\left(H + \frac{2}{5}\right)$$

由 \(\frac{P_1}{P_2} = \frac{5}{4}\)，得 \(3H^2 - 5H - 2 = 0\)，解得 \(H = -\frac{1}{3}\) (舍去)，\(H = 2\)。

设有一长度为 l，线密度为常数 μ 的细棒，在细棒右端距离为a处有一质量为m的质点M (见图)，已知引力常量为G，则质点M与细棒之间的引力的大小为：

$$\int_d^b \frac{Gm\mu}{(a-x)^2}dx$$
例12.5

设沿y轴上的区间 [0,1] 放置一长度为1且线密度为ρ的均匀细杆，在x轴上x = 1处有一单位质点，则该细杆对此质点x轴正向的分力为_______。

【解】应填 \(-\frac{\sqrt{2}}{2}G\rho\)。

依题意画出直角坐标系如图所示。用割元法，取y轴上的 [y, y+dy] ⊂ [0,1]，此微段细杆对质点的引力大小为：

$$dF = G\frac{\rho dy}{1+y^2}$$

其在x轴正向的分力为

$$dF_x = \frac{-1}{\sqrt{1+y^2}}dF = -\frac{G\rho dy}{(1+y^2)^{\frac{3}{2}}}$$

于是该细杆对此质点沿x轴正向的分力为

$$F_x = \int_0^1 \left[-\frac{G\rho}{(1+y^2)^{\frac{3}{2}}}\right]dy \overset{\angle y=\tan t}{=} -G\rho\int_0^{\frac{\pi}{4}} \cos tdt = -\frac{\sqrt{2}}{2}G\rho$$

（D₁（常规操作）+D₂₂（转换等价表述））

例12.6

水从一根底半径为1 cm的圆柱形管道中流出，因为水有黏性，在流动过程中受到管道壁的阻滞，所以流动的速度是随着看到管道中心的距离而变化的，距管道中心越远，水流速度越小。在距离管道中心 r cm 处的水的流动速度为 \(10(1-r^2)\) cm/s。问水是以多大流量(以 cm³/s为单位)流过管道的？

【解】

取 [r, r+dr] ⊂ [0,1]，此圆环中的水流速度为 \(10(1-r^2)\) cm/s。该圆环的载面积约等于 \(2\pi rdr\)。

于是单位时间内通过该圆环的水量为 \(dQ = 10(1-r^2) \cdot 2\pi rdr\)。单位时间内通过整个圆柱形管道的水量为

$$Q = \int_0^1 20\pi r(1-r^2)dr = \pi(10r^2 - 5r^4)\Big|_0^1 = 5\pi(\text{cm}^3)$$

于是，水流过管道的流量为 5π cm³/s。

第二部分：求解经济应用题（仅数学三）

三问解题法

求解经济应用题（仅数学三）
（O(Ⅱ)任目标))
↓
总成本函数：
\(C(Q) = \int_0^Q C'(t)dt + C_0\);

总收益函数：
\(R(Q) = \int_0^Q R'(t)dt\);

总利润函数：
\(L(Q) = R(Q) - C(Q)\)
\(= \int_0^Q [R'(t) - C'(t)]dt - C_0\)

已知一个总函数(如总成本函数、总收益函数等)，利用微分或求导算就可以求出其边际函数(边际成本、边际收益等)。反过来，如果已知边际函数，要确定其总函数就要利用积分运算。

当固定成本为 C₀，边际成本为 C'(Q)，边际收益为 R'(Q)，且产销平衡，即产量、需求量与销量均为Q时：

总成本函数

$$C(Q) = \int_0^Q C'(t)dt + C_0$$

总收益函数

$$R(Q) = \int_0^Q R'(t)dt$$

总利润函数

$$L(Q) = R(Q) - C(Q) = \int_0^Q [R'(t) - C'(t)]dt - C_0$$
例12.7

某厂生产的产品的边际成本为产量 x 的函数 \(C'(x) = x^2 - 4x + 6\)，固定成本为 \(C_0 = 200\) 千元，且每单位产品的售价为 \(p = 146\) 千元，并假定生产的产品能全部售出。

(1) 求总成本函数 C(x)；

(2) 求产量从2个单位增加到4个单位时的成本变化量(计算结果保留两位小数)；

(3) 当产量为多大时，总利润最大？并求最大利润(计算结果保留两位小数)。

【解】(1) 总成本函数：

$$C(x) = \int_0^x C'(t)dt + C_0 = \int_0^x (t^2 - 4t + 6)dt + 200$$

$$= \frac{x^3}{3} - 2x^2 + 6x + 200$$

(2) 设产量由2个单位增加到4个单位时的成本变化量为 ΔC，则

$$\Delta C = \int_2^4 C'(x)dx = \int_2^4 (x^2 - 4x + 6)dx$$

$$= \left(\frac{x^3}{3} - 2x^2 + 6x\right)\Big|_2^4$$

$$= \frac{20}{3} \approx 6.67(\text{千元})$$

或：$$\Delta C = C(4) - C(2) = \frac{20}{3} \approx 6.67(\text{千元})$$

(3) 总收益函数为：

$$R(x) = p \cdot x = 146x$$

总利润函数为：

$$L(x) = R(x) - C(x)$$

$$= 146x - \left(\frac{x^3}{3} - 2x^2 + 6x + 200\right)$$

$$= -\frac{x^3}{3} + 2x^2 + 140x - 200$$

故：$$L'(x) = -x^2 + 4x + 140$$

令 \(L'(x) = 0\)，得 \(x_1 = 14\)，\(x_2 = -10\) (舍去)。

由于 \(L'(x) = -2x + 4\)，所以 \(L'(14) = -24 < 0\)，因此，当 \(x = 14\) 时，总利润最大，最大利润为

$$L_{\max} = L(14) = \frac{3712}{3} \approx 1237.33(\text{千元})$$

即当产量为14个单位时，利润达到最大，最大利润是1237.33千元。
