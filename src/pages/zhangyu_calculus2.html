📚 核心思路概述

判断数列 \(\{x_n\}\) 是否收敛（即 \(\lim_{n\to\infty} x_n\) 是否存在）有以下五种主要方法：

1️⃣ 见到 \(f(x, x_n)\)，且其是含 \(x_n, x_{n+1}\) 的等式关系
2️⃣ 见到 \(f(x, x_{n+1})\)，且其是含 \(x_n, x_{n+1}\) 的不等式关系
3️⃣ 初值 \(x_1\) 对 \(x_{n+1} = f(x_n)\) 的敛散性的影响
4️⃣ 双通项 \(a_n, b_n\) 问题
5️⃣ 复合函数的极限

方法一：等式关系 + 常规操作 + 化归经典形式

📐 压缩映射原理

原理一： 对数列 \(\{x_n\}\)，若存在常数 \(k\) (\(0 < k < 1\))，使得 \(|x_{n+1}-a| \leq k|x_n-a|\), \(n=1,2,\cdots\)，则 \(\{x_n\}\) 收敛于 \(a\)。

证明： \(0 \leq |x_{n+1}-a| \leq k|x_n-a| \leq k^2|x_{n-1}-a| \leq \cdots \leq k^n|x_1-a|\)

由于 \(\lim_{n\to\infty} k^n = 0\)，根据夹逼准则，有 \(\lim_{n\to\infty}|x_{n+1}-a|=0\)，即 \(\{x_n\}\) 收敛于 \(a\)。

原理二： 对数列 \(\{x_n\}\)，若 \(x_{n+1} = f(x_n)\), \(n=1,2,\cdots\), \(f(x)\) 可导，\(a\) 是 \(f(x)=x\) 的唯一解，且对任意 \(x\in\mathbb{R}\)，有 \(|f'(x)| \leq k < 1\)，则 \(\{x_n\}\) 收敛于 \(a\)。

🔢 例2.1

若 \(x_1=1\), \(x_{n+1}=\sqrt{4+3x_n}\), \(n=1,2,\cdots\)，证明数列 \(\{x_n\}\) 收敛，并求 \(\lim_{n\to\infty} x_n\)。

💡 解答

令 \(f(x) = \sqrt{4+3x}\) (\(x>0\))，则 \(f(4)=4\)，且

\[|f'(x)| = \frac{3}{2\sqrt{4+3x}} \leq \frac{3}{2\sqrt{4}} = \frac{3}{4} < 1\]

又有

\[|x_{n+1}-4| = |f(x_n)-f(4)| = |f'(\xi)||x_n-4| \leq \frac{3}{4}|x_n-4|\]

其中 \(\xi\) 介于 \(x_n\) 与 4 之间，从而有

\[0 < |x_{n+1}-4| \leq \frac{3}{4}|x_n-4| \leq \left(\frac{3}{4}\right)^2|x_{n-1}-4| \leq \cdots \leq \left(\frac{3}{4}\right)^n|x_1-4|\]

而当 \(n\to\infty\) 时，\(\left(\frac{3}{4}\right)^n \to 0\)，故数列 \(\{x_n\}\) 收敛，且 \(\lim_{n\to\infty} x_n = 4\)。

🔢 例2.2

若 \(x_{n+1}=\frac{\pi}{2}+\frac{1}{2}\cos x_n\) (\(n=1,2,\cdots\)), \(x_1=\pi\)，证明数列 \(\{x_n\}\) 收敛，并求 \(\lim_{n\to\infty} x_n\)。

💡 解答

令 \(f(x)=\frac{\pi}{2}+\frac{1}{2}\cos x\)，则 \(f\left(\frac{\pi}{2}\right)=\frac{\pi}{2}\)，\(|f'(x)|=\left|-\frac{1}{2}\sin x\right| \leq \frac{1}{2} < 1\)，则有

\[\left|x_{n+1}-\frac{\pi}{2}\right| = \left|f(x_n)-f\left(\frac{\pi}{2}\right)\right| \leq \frac{1}{2}\left|x_n-\frac{\pi}{2}\right|\]

从而有

\[\left|x_{n+1}-\frac{\pi}{2}\right| \leq \frac{1}{2}\left|x_n-\frac{\pi}{2}\right| \leq \cdots \leq \left(\frac{1}{2}\right)^n\left|x_1-\frac{\pi}{2}\right|\]

又当 \(n\to\infty\) 时，\(\left(\frac{1}{2}\right)^n \to 0\)，故数列 \(\{x_n\}\) 收敛，且 \(\lim_{n\to\infty} x_n = \frac{\pi}{2}\)。

方法二：不等式关系（常规操作）

比较 \(x_n, x_{n+1}\) 的大小，定单调，根据正定单调有界准则。

🔢 例2.3

若 \(0 < x_n < 1\), \(x_{n+1}=1-\sqrt{1-x_n}\) (\(n=1,2,\cdots\))，求：

(1) \(\lim_{n\to\infty} x_n\)；

(2) \(\lim_{n\to\infty} \frac{x_{n+1}}{x_n}\)。

💡 解答

(1) 由 \(x_{n+1}=1-\sqrt{1-x_n}=\frac{x_n}{1+\sqrt{1-x_n}} < x_n\), \(0 < x_n < 1\)，知数列 \(\{x_n\}\) 单调减少且有下界，故数列 \(\{x_n\}\) 收敛。

设 \(\lim_{n\to\infty} x_n = a\)，则有 \(a=1-\sqrt{1-a}\)，解得 \(a=0\) (\(a=1\) 舍去)，故 \(\lim_{n\to\infty} x_n = 0\)。

(2) 由 (1) 可知，\(\lim_{n\to\infty} \frac{x_{n+1}}{x_n} = \lim_{n\to\infty} \frac{1-\sqrt{1-x_n}}{x_n} = \lim_{n\to\infty} \frac{1}{1+\sqrt{1-x_n}} = \frac{1}{2}\)。

🔢 例2.4

设 \(x_{n+1}=\frac{x_n^2}{2(x_n-1)}\) (\(n=1,2,\cdots\)), \(x_1>1\)，证明数列 \(\{x_n\}\) 收敛，并求 \(\lim_{n\to\infty} x_n\)。

💡 解答

\[x_{n+1}=\frac{1}{2}\left(x_n-1+\frac{1}{x_n-1}\right)+1 \geq 2, x_{n+1}-x_n=\frac{x_n(2-x_n)}{2(x_n-1)} \leq 0\]

故由单调有界准则知，数列 \(\{x_n\}\) 收敛。

设 \(\lim_{n\to\infty} x_n = A\)，则有 \(A=\frac{A^2}{2(A-1)}\)，解得 \(A=2\) 或 \(A=0\) (舍去)，故 \(\lim_{n\to\infty} x_n = 2\)。

方法三：初值对敛散性的影响（常规操作）

对于 \(x_{n+1} = f(x_n)\)，若初值 \(x_1\) 仅给定取值范围，可能需要分情况讨论。

🔢 例2.5

设 \(x_{n+1}=\sqrt{x_n(2-x_n)}\) (\(n=1,2,\cdots\)), \(0 < x_1 < 2\)，证明数列 \(\{x_n\}\) 的极限存在，并求此极限。

💡 解答

首先证明数列 \(\{x_n\}\) 有界。因为 \(0 < x_1 < 2\)，所以 \(x_1, 2-x_1\) 均为正数，从而

\[0 < x_2 = \sqrt{x_1(2-x_1)} \leq \frac{x_1+2-x_1}{2} = 1\]

设 \(0 < x_k \leq 1 (k>1)\)，则 \(0 < x_{k+1} = \sqrt{x_k(2-x_k)} \leq \frac{1}{2}(x_k+2-x_k)=1\)，由数学归纳法知，对任意正整数 \(n>1\)，都有 \(0 < x_n \leq 1\)，即数列 \(\{x_n\}\) 有界。

再证明数列 \(\{x_n\}\) 单调。当 \(n>1\) 时，

\[\frac{x_{n+1}}{x_n} = \sqrt{\frac{2}{x_n}-1} \geq 1\]

即 \(x_{n+1} \geq x_n (n>1)\)，所以数列 \(\{x_n\}\) (\(n>1\)) 单调增加。

根据单调有界准则知 \(\lim_{n\to\infty} x_n\) 存在，设其为 \(a\)，则

\[a = \lim_{n\to\infty} x_n = \lim_{n\to\infty} \sqrt{x_{n-1}(2-x_{n-1})} = \sqrt{a(2-a)}\]

解得 \(a=1\) 或 \(a=0\) (舍去)，故 \(\lim_{n\to\infty} x_n = 1\)。

🔢 例2.6

(1) 证明对任意正整数 \(n\)，都有 \(\frac{1}{n+1} < \ln\left(1+\frac{1}{n}\right) < \frac{1}{n}\) 成立；

(2) 设 \(a_n=1+\frac{1}{2}+\cdots+\frac{1}{n}-\ln n\) (\(n=1,2,\cdots\))，证明数列 \(\{a_n\}\) 收敛。

💡 解答

(1) 令 \(f(x)=\ln x\) (\(x>0\))。对任意正整数 \(n\)，对 \(f(x)\) 在 \([n,n+1]\) 上使用拉格朗日中值定理，得

\[\ln\left(1+\frac{1}{n}\right) = \ln(1+n)-\ln n = \frac{1}{\xi}\]

其中 \(n < \xi < n+1\)，所以 \(\frac{1}{n+1} < \ln\left(1+\frac{1}{n}\right) < \frac{1}{n}\)。

(2) 由 (1) 知，当 \(n\geq 1\) 时，有

\[a_{n+1}-a_n = \frac{1}{n+1}-\ln\left(1+\frac{1}{n}\right) < 0\]

\[a_n = 1+\frac{1}{2}+\cdots+\frac{1}{n}-\ln n > \ln(1+1)+\ln\left(1+\frac{1}{2}\right)+\cdots+\ln\left(1+\frac{1}{n}\right)-\ln n = \ln(1+n)-\ln n > 0\]

故数列 \(\{a_n\}\) 单调减少且有下界，所以数列 \(\{a_n\}\) 收敛。

方法四：双通项 \(a_n, b_n\) 问题

策略：

(1) 将 \(a_n, b_n\) 满足的式子联立，消去 \(a_n\) 或 \(b_n\) 其中一个，转化为单通项问题，且常用①恒等变形；②无穷小比较；③常见放缩；④函数单调，来作细节处理。

(2) 令 \(c_n=\frac{a_n}{b_n}\)，转求 \(\lim_{n\to\infty} c_n\)，且常用①单调有界准则；②夹逼准则；③极限保号性，来作细节处理。

🔢 例2.9

设数列 \(\{a_n\}\), \(\{b_n\}\) 满足 \(a_n+b_n=\frac{\pi}{2}\), \(a_{n+1}=a_n+\cos a_n\)，其中 \(n=1,2,\cdots\), \(a_1=1\)，当 \(n\to\infty\) 时，\(b_n\) 与 \(b_{n-1}\) 为同阶无穷小量，则 \(a=\)（    ）。

(A) 1    (B) 2    (C) 3    (D) 5

💡 解答

应选 (C)。

\[b_n = \frac{\pi}{2}-a_n = \frac{\pi}{2}-a_{n-1}-\cos a_{n-1}\]

\[= \frac{\pi}{2}-a_{n-1}-\sin\left(\frac{\pi}{2}-a_{n-1}\right)\]

\[= b_{n-1}-\sin b_{n-1}(n\geq 2)\]

由 \(a_1=1\)，则 \(b_1=\frac{\pi}{2}-1\in(0,1), b_2=b_1-\sin b_1 < b_1\)，即 \(0 < b_2 < b_1 < 1\)。

假设当 \(n=k(k\geq 2)\) 时，\(0 < b_k < b_{k-1} < 1\) 成立，则当 \(n=k+1\) 时，

\[0 < b_{k+1} = b_k-\sin b_k < b_k < 1\]

故 \(\{b_n\}\) 单调有界，则由单调有界准则，知 \(\{b_n\}\) 收敛。令 \(\lim_{n\to\infty} b_n = A\)，在等式 \(b_n=b_{n-1}-\sin b_{n-1}\) 两端取极限，得 \(A=A-\sin A\)，故 \(A=0\)，即 \(\lim_{n\to\infty} b_n = 0\)。

故当 \(n\to\infty\) 时，\(b_n\to 0\) 时，\(b_n=b_{n-1}-\sin b_{n-1}\sim \frac{1}{6}b_{n-1}^3\)，因此 \(a=3\)。

方法五：复合函数的极限

定理一（因变量极限定理）

设 \(y=f[g(x)]\), \(u=g(x)\), \(y=f(u)\)。若：

① \(\lim_{x\to x_0} g(x) = u_0\)

② \(\lim_{u\to u_0} f(u) = a\)

③ 当 \(x\neq x_0\) 时，\(g(x)\neq u_0\)

则 \(\lim_{x\to x_0} f[g(x)] = a\)。

注意： 关键是当 \(x\neq 0\) 时，\(g(x)\) 也有零点，故复合函数的极限定理一的这算并不一定成立。

定理二（中间变量极限定理）

设 \(u_n\in\) 有限区间 \(I\)，若 \(f(x)\) 在 \(I\) 上严格单调，且 \(\lim_{n\to\infty} f(u_n)\) 存在，则 \(\lim_{n\to\infty} u_n\) 存在。

🔢 例2.12

已知数列 \(\{x_n\}\)，其中 \(-\frac{\pi}{2}\leq x_n\leq\frac{\pi}{2}\)，则（    ）。

(A) 若 \(\lim_{n\to\infty}\cos\sin x_n\) 存在，则 \(\lim_{n\to\infty} x_n\) 存在

(B) 若 \(\lim_{n\to\infty}\sin\cos x_n\) 存在，则 \(\lim_{n\to\infty} x_n\) 存在

(C) 若 \(\lim_{n\to\infty}\cos\sin x_n\) 存在，则 \(\lim_{n\to\infty}\sin x_n\) 存在，但 \(\lim_{n\to\infty} x_n\) 不一定存在

(D) 若 \(\lim_{n\to\infty}\sin\cos x_n\) 存在，则 \(\lim_{n\to\infty}\cos x_n\) 存在，但 \(\lim_{n\to\infty} x_n\) 不一定存在

💡 解答

应选 (D)。

根据定理二，由于 \(\sin x\) 在 \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\) 上单调，因此，由 \(\lim_{n\to\infty}\cos\sin x_n\) 存在可得 \(\lim_{n\to\infty}\sin x_n\) 存在，但 \(\cos x\) 在 \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\) 上不单调，因此，由 \(\lim_{n\to\infty}\cos\sin x_n\) 存在无法得到 \(\lim_{n\to\infty}\cos x_n\) 存在，进而无法得到 \(\lim_{n\to\infty} x_n\) 存在。故 (A) 正确，(B) 错误。

由于 \(\cos x\) 在 \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\) 上不单调，因此，由 \(\lim_{n\to\infty}\cos\sin x_n\) 存在无法得到 \(\lim_{n\to\infty}\sin x_n\) 存在，但可知 (D) 正确，(C) 错误。
