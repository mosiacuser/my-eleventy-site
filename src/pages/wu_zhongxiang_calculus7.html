一、常微分方程的基本概念

1. 微分方程

含有未知函数的导数或微分的方程称为微分方程，简称方程。

2. 微分方程的阶

微分方程中所出现的未知函数最高阶导数的阶数，称为微分方程的阶。

3. 微分方程的解

满足微分方程的函数，称为该方程的解。

4. 微分方程的通解

如果微分方程的解中含有任意常数，且任意常数与微分方程的阶数相同，则称之为微分方程的通解。

5. 微分方程的特解

微分方程的不含任意常数的解，称之为特解。

6. 初始条件

确定特解的一组常数称为初始条件。

积分曲线：方程的一个解在平面上对应一条曲线，称为该微分方程的积分曲线。

二、一阶微分方程

1. 可分离变量的方程

能表示为 \(g(y)dy = f(x)dx\) 的方程称为可分离变量的方程。

求解方法是两端积分： $$\int g(y)dy = \int f(x)dx$$
例题 1 (2006，数二)：微分方程 \(y' = \frac{y(1-x)}{x}\) 的通解是？

分离变量：

$$\frac{dy}{dx} = \frac{y(1-x)}{x}$$ $$\frac{dy}{y} = \frac{1-x}{x}dx$$ $$\int\frac{dy}{y} = \int\frac{1-x}{x}dx$$ $$\ln|y| = \ln|x| - x + C_1$$ $$y = Cxe^{-x}$$

其中 \(C\) 为任意常数。

2. 齐次微分方程

能化为 \(\frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)\) 的方程称为齐次微分方程。

求解齐次微分方程的标准方法：令 \(u = \frac{y}{x}\)，则 \(y = ux\)，代入后化为可分离变量方程。
$$xu' = \varphi(u) - u$$
例题 2 (1993，数二)：求微分方程 \(x^2y' + xy = y^2\) 满足初始条件 \(y(1) = 1\) 的特解。
3. 一阶线性微分方程

形如 \(y' + p(x)y = q(x)\) 的方程称为一阶线性微分方程。

通解公式（常数变易法）： $$y = e^{-\int p(x)dx}\left[\int q(x)e^{\int p(x)dx}dx + C\right]$$
例题 3 (2008，数二、四)：微分方程 \((y + x^2e^{-x})dx - xdy = 0\) 的通解是？

原方程可改写为：

$$\frac{dy}{dx} - \frac{1}{x}y = xe^{-x}$$

由线性方程通解公式得：

$$y = e^{\int\frac{1}{x}dx}\left[\int xe^{-x}e^{-\int\frac{1}{x}dx}dx + C\right]$$ $$= x\left[\int e^{-x}dx + C\right]$$ $$= x(C - e^{-x})$$

其中 \(C\) 为任意常数。

4. 伯努利方程

形如 \(y' + p(x)y = Q(x)y^n\) (\(n \neq 0, 1\)) 的方程称为伯努利方程。

求解伯努利方程的方法：令 \(u = y^{1-n}\)，将原方程化为一阶线性微分方程。
5. 全微分方程

如果方程 \(P(x,y)dx + Q(x,y)dy = 0\) 是某个函数 \(u(x,y)\) 的全微分：

$$du(x,y) = P(x,y)dx + Q(x,y)dy$$

则称该方程为全微分方程。此方程的通解为 \(u(x,y) = C\)。

充要条件：当 \(P(x,y)\)，\(Q(x,y)\) 在连通域 \(G\) 内具有一阶连续偏导数时，方程为全微分方程的充要条件是： $$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}, \quad (x,y) \in G$$

三、可降阶的高阶微分方程

数学三不要求

1. \(y^{(n)} = f(x)\) 型的微分方程

连续积分 \(n\) 次即可。

2. \(y'' = f(x, y')\) 型的方程

令 \(y' = p\)，则 \(y'' = p'\)，原方程化为一阶方程。

3. \(y'' = f(y, y')\) 型的方程

令 \(y' = p\)，则 \(y'' = p\frac{dp}{dy}\)，原方程化为一阶方程。

例题 5 (2002，数一、二)：求微分方程 \(yy'' + y'^2 = 0\) 满足初始条件 \(y|_{x=0} = 1, y'|_{x=0} = \frac{1}{2}\) 的特解。

令 \(y' = p\)，则 \(y'' = p\frac{dp}{dy}\)，代入原方程得：

$$yp\frac{dp}{dy} + p^2 = 0$$ $$y\frac{dp}{dy} = -p$$

分离变量积分得：

$$\ln|yp| = C_1$$ $$yp = C$$

由初始条件得 \(C = \frac{1}{2}\)，所以：

$$y\frac{dy}{dx} = \frac{1}{2}$$ $$\int ydy = \frac{1}{2}\int dx$$ $$\frac{1}{2}y^2 = \frac{1}{2}x + C_2$$

由 \(y|_{x=0} = 1\) 得 \(C_2 = \frac{1}{2}\)，故特解为：

$$y = \sqrt{x + 1}$$

四、高阶线性微分方程

1. 线性微分方程解的结构

二阶线性微分方程的一般形式为：

$$y'' + p(x)y' + q(x)y = f(x)$$
齐次方程：\(y'' + p(x)y' + q(x)y = 0\)
非齐次方程：\(y'' + p(x)y' + q(x)y = f(x)\)
定理 内容
定理1 如果 \(y_1(x)\) 和 \(y_2(x)\) 是齐次方程的两个线性无关的特解，那么 \(y = C_1y_1(x) + C_2y_2(x)\) 就是方程的通解。
定理2 如果 \(y^*(x)\) 是非齐次方程的一个特解，\(y_1(x)\)、\(y_2(x)\) 是齐次方程的两个线性无关的特解，则 \(y = C_1y_1(x) + C_2y_2(x) + y^*(x)\) 是非齐次微分方程的通解。
定理3 如果 \(y_1^*(x)\)，\(y_2^*(x)\) 是非齐次方程的两个特解，则 \(y(x) = y_1^*(x) - y_2^*(x)\) 是齐次微分方程的解。

2. 常系数齐次线性微分方程

常系数齐次微分方程的一般形式为：

$$y'' + py' + qy = 0$$

其特征方程为：\(r^2 + pr + q = 0\)

通解形式：
  • (1) 若 \(r_1 \neq r_2\) 为两个不相等的实特征根，通解为 \(y = C_1e^{r_1x} + C_2e^{r_2x}\)
  • (2) 若 \(r_1 = r_2\) 为二重实特征根，通解为 \(y = (C_1 + C_2x)e^{r_1x}\)
  • (3) 若 \(r_{1,2} = \alpha \pm i\beta\) 为复根，通解为 \(y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)\)
例题 7 (1996，数三)：微分方程 \(y'' + 2y' + 5y = 0\) 的通解为？

特征方程：\(r^2 + 2r + 5 = 0\)

特征根：\(r_{1,2} = -1 \pm 2i\)

因此通解为：

$$y = e^{-x}(C_1\cos 2x + C_2\sin 2x)$$

3. 常系数非齐次线性微分方程

非齐次微分方程的一般形式为：

$$y'' + py' + qy = f(x)$$
特解设定规则：
  • (1) 若 \(f(x) = P_m(x)e^{kx}\)，则特解 \(y^* = x^kQ_m(x)e^{kx}\)，其中 \(k\) 是特征根 \(k\) 的重复次数
  • (2) 若 \(f(x) = e^{\alpha x}[P_l^{(1)}(x)\cos\beta x + P_n^{(2)}(x)\sin\beta x]\)，则特解 \(y^* = x^ke^{\alpha x}[R_m^{(1)}(x)\cos\beta x + R_m^{(2)}(x)\sin\beta x]\)，其中 \(m = \max(l,n)\)

五、差分方程

数学三要求

1. 一阶常系数线性齐次差分方程

$$y_{t+1} + ay_t = 0$$

通解为：\(y_t = C \cdot (-a)^t\)，\(C\) 为任意常数。

2. 一阶常系数线性非齐次差分方程

$$y_{t+1} + ay_t = f(t)$$

通解为：\(y_t = y_c(t) + y^*\)，其中 \(y^*\) 是非齐次差分方程的特解。

六、典型例题与应用

例题 14 (2001，数三)：某公司每年的工资总额在比上一年增加20%的基础上再追加2百万元，若以 \(W_t\) 表示第 \(t\) 年的工资总额（单位：百万元），则 \(W_t\) 满足的差分方程是？

根据题意：

$$W_t = 1.2 \cdot W_{t-1} + 2$$
例题 16 (2014，数一)：微分方程 \(xy' + y(\ln x - \ln y) = 0\) 满足条件 \(y(1) = e^3\) 的解为？

这是齐次方程。令 \(u = \frac{y}{x}\)，代入后：

$$u' = \frac{u(\ln u - 1)}{x}$$

分离变量积分：

$$\frac{du}{u(\ln u - 1)} = \frac{dx}{x}$$ $$\ln|\ln u - 1| = \ln x + C_1$$

由 \(y(1) = e^3\) 得 \(C = 2\)，因此：

$$\ln\frac{y}{x} - 1 = 2x$$ $$y = xe^{2x+1}$$
