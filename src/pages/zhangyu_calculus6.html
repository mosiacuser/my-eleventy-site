找到"是谁"求导，得到欲证结论，也即 \([F(x)]'=0\)，从而应用罗尔定理，或研究 \(F(x)\) 的性态。

1. 一阶乘积求导公式的逆用（化归经典形式）

$$(uv)' = u'v + uv'$$

常见形式包括：

$$①[f(x)·x^n]' = x^{n-1}[xf'(x) + nf(x)]$$ $$②[f(x)e^{ax}]' = e^{ax}[f'(x) + af(x)]$$ $$③[f(x)e^x]' = e^x[f'(x) + f(x)]$$ $$④[f(x)e^{p(x)}]' = e^{p(x)}[f'(x) + f(x)p'(x)]$$
📘 例6.1

设函数 \(f(x)\) 在 \([0,1]\) 上连续，\((0,1)\) 内可导，\(f(0)=f(1)\)，\(\int_0^1 f(x)dx=0\)，证明：存在 \(\xi \in (0,1)\)，使得 \(f'(\xi)+f^2(\xi)=0\)。

【证】令 \(F(x)=f(x)e^{\int_0^x f(t)dt}\)，则

$$F(0) = f(0)e^0 = f(0), \quad F(1) = f(1)e^{\int_0^1 f(t)dt} = f(1)$$

即 \(F(0)=F(1)\)，由罗尔定理，存在 \(\xi \in (0,1)\)，使得 \(F'(\xi)=0\)，即

$$e^{\int_0^\xi f(t)dt}[f'(\xi) + f^2(\xi)] = 0$$

得证。

2. 二阶乘积求导公式的逆用（化归经典形式）

$$(uv)'' = u''v + 2u'v' + uv''$$
$$[f(x)·e^x]'' = e^x[f''(x) + 2f'(x) + f(x)]$$

📘 例6.7

设函数 \(f(x)\) 在 \([0,1]\) 上二阶可导，\(f(0)=0\)，且 \(f(x)\) 在 \((0,1)\) 内取得最大值2，在 \((0,1)\) 内取得最小值，证明：

(1) 存在 \(\xi \in (0,1)\)，使得 \(f'(\xi) > 2\)；

(2) 存在 \(\eta \in (0,1)\)，使得 \(f''(\eta) < -4\)。

【证】(1) 设 \(f(x)\) 在 \(x_1 \in (0,1)\) 处取得最大值，即 \(f(x_1)=2\)。在 \([0,x_1]\) 上对 \(f(x)\) 应用拉格朗日中值定理，得

$$f(x_1) - f(0) = f'(\xi)(x_1-0), \quad \xi \in (0,x_1) \subset (0,1)$$

即 \(f'(\xi)·x_1 = 2\)。又 \(0 < x_1 < 1\)，则 \(\frac{1}{x_1} > 1\)，即 \(f'(\xi) = \frac{2}{x_1} > 2\)。

第二部分 讨论 \(f(x)=0\) 的根的个数

讨论 \(f(x)=0\) 的根的个数
(O(Ⅲ作目标))

⬇️

反证思想
罗尔定理的推论
渐近性态

设 \(f(x)\) 在 \([a,b]\) 上连续，且 \(f(a)f(b) < 0\)，则 \(f(x)=0\) 在 \((a,b)\) 内至少有一个根。

📌 推广

推广的零点定理：若 \(f(x)\) 在 \((a,b)\) 内连续，\(\lim_{x \to a^+}f(x) = \alpha\)，\(\lim_{x \to b^-}f(x) = \beta\)，且 \(\alpha \beta < 0\)，则 \(f(x)=0\) 在 \((a,b)\) 内至少有一根。这里 \(a, b, \alpha, \beta\) 可以是有限数，也可以是无穷大。

📘 例6.17

设函数 \(f(x) = ax - b\ln x(a>0)\) 有2个零点，则 \(\frac{b}{a}\) 的取值范围是（　）。

(A) \((0,e)\) 　(B) \((e,+\infty)\) 　(C) \(\left(0,\frac{1}{e}\right)\) 　(D) \(\left(\frac{1}{e},+\infty\right)\)

【解】应选(B)。

由 \(f(x) = ax - b\ln x(a>0)\)，则 \(x>0\) 且 \(f'(x) = a - \frac{b}{x}\)。当 \(b \leq 0\) 时，\(f'(x) \geq 0\)，不满足条件，舍去；

当 \(b>0\) 时，令 \(f'(x)=0\)，得 \(x=\frac{b}{a}\)。当 \(x \in \left(0,\frac{b}{a}\right)\) 时，\(f'(x) < 0\)；当 \(x \in \left(\frac{b}{a},+\infty\right)\) 时，\(f'(x) > 0\)。又

$$\lim_{x \to 0^+}f(x) = +\infty, \quad \lim_{x \to +\infty}f(x) = +\infty$$

则应有 \(f\left(\frac{b}{a}\right) = b - b\ln\frac{b}{a} = b\left(1-\ln\frac{b}{a}\right) < 0\)，得 \(\ln\frac{b}{a} > 1\)，即 \(\frac{b}{a} > e\)。故选(B)。

若证 \(f(x)\) 存在零点，可设 \(f(x)\) 无零点，按条件推证出矛盾；若证 \(f(x)\) 无零点，可设 \(f(x)\) 存在零点，按条件推证出矛盾。

通过研究函数的导数，判断函数的单调性、极值点等性态，从而确定零点个数。

第三部分 证明不等式

证明不等式
(O(Ⅲ作目标))

⬇️

用单调性
用凹凸性
用柯西中值定理

1. 用单调性

(1) 若 \(\lim_{x \to a^+}F(x) \geq 0\)，且当 \(x \in (a,b)\) 时 \(F'(x) \geq 0\)，则在 \((a,b)\) 内 \(F(x) \geq 0\)。

(2) 若 \(\lim_{x \to b^-}F(x) \geq 0\)，且当 \(x \in (a,b)\) 时 \(F'(x) \leq 0\)，则在 \((a,b)\) 内 \(F(x) \geq 0\)。

💡 注意

(1) 若在 \(x=a\) 处 \(F(x)\) 右连续，则可用 \(F(a)\) 代替 \(\lim_{x \to a^+}F(x)\)。

(2) 若当 \(x \in (a,b)\) 时，\(F'(x) > 0\)，则在 \((a,b)\) 内 \(F(x) > 0\)。

2. 用最值

如果在 \((a,b)\) 内 \(F(x)\) 有最小值 \(m\)，则在 \((a,b)\) 内 \(F(x) \geq m\)，且除这些最小值点外，均有 \(F(x) > m\)。

3. 用凹凸性

如果对于任意的 \(x \in I\)，\(F''(x) > 0\)，则

① 对于任意的 \(x_1, x_2 \in I\)，有

$$\frac{F(x_1)+F(x_2)}{2} \geq F\left(\frac{x_1+x_2}{2}\right)$$

② 对于任意的 \(x_1, x_2 \in I\)，任意的 \(\lambda_1, \lambda_2 \in (0,1)\)，且 \(\lambda_1 + \lambda_2 = 1\)，有

$$\lambda_1 F(x_1) + \lambda_2 F(x_2) \geq F(\lambda_1 x_1 + \lambda_2 x_2)$$
📘 例6.19

设函数 \(f(x)\) 在区间 \((a, b)\) 内可导，证明：导数 \(f'(x)\) 在 \((a,b)\) 内严格单调递增的充分必要条件是对 \((a, b)\) 内任意的 \(x_1, x_2, x_3\)，当 \(x_1 < x_2 < x_3\) 时，\(\frac{f(x_2)-f(x_1)}{x_2-x_1} < \frac{f(x_3)-f(x_2)}{x_3-x_2}\)。

第四部分 求解含参等式或不等式问题

求解含参等式或不等式问题
(O(Ⅲ作目标))

⬇️

导数中不含参数
研究函数性态
导数中含参数
研究曲线交点

(1) 导数中不含参数

辅助函数 \(f'(x)\) 中不含参数，于是研究函数性态的过程中不讨论参数，结果中讨论参数，即根据参数的取值不同，研究曲线与x轴的位置关系。

(2) 导数中含参数

辅助函数 \(f'(x)\) 中含参数，于是研究函数性态的过程中讨论参数，即根据参数的取值不同，研究曲线不同的性态，从而确定其与x轴的交点个数。

📘 例6.20

已知常数 \(k \geq \ln 2 - 1\)，证明：\((x-1)(x-\ln^2 x + 2k\ln x - 1) \geq 0\)。

【证】设 \(f(x) = x - \ln^2 x + 2k\ln x - 1(x>0)\)，则

$$f'(x) = 1 - \frac{2\ln x}{x} + \frac{2k}{x} = \frac{1}{x}(x - 2\ln x + 2k)$$

设 \(g(x) = x - 2\ln x + 2k\)，则 \(g'(x) = 1 - \frac{2}{x}\)。令 \(g'(x)=0\)，得 \(g(x)\) 的唯一驻点 \(x=2\)。

又 \(g''(x) = \frac{2}{x^2} > 0\)，故 \(x=2\) 为 \(g(x)\) 的唯一极小值点，于是 \(g(2)\) 为 \(g(x)\) 的最小值。

因为已知 \(k \geq \ln 2 - 1\)，所以 \(g(2) = 2 - 2\ln 2 + 2k \geq 0\)，从而 \(g(x) \geq 0\)。

综上可知 \(f'(x) \geq 0\)，所以 \(f(x)\) 单调递增。

📊 泰勒展开式速查表

第一组

$$\sqrt{1\pm x} = 1 \pm \frac{1}{2}x - \frac{1}{8}x^2 + \cdots, \quad |x| < 1$$ $$\frac{1}{\sqrt{1\pm x}} = 1 \mp \frac{1}{2}x + \frac{3}{8}x^2 + \cdots, \quad |x| < 1$$ $$\frac{1}{1+x} = 1 - x + x^2 + \cdots = \sum_{n=0}^{\infty}(-1)^n x^n, \quad |x| < 1$$ $$\frac{1}{1-x} = 1 + x + x^2 + \cdots = \sum_{n=0}^{\infty}x^n, \quad |x| < 1$$

第二组（指数和双曲函数）

$$e^x = \sum_{n=0}^{\infty}\frac{x^n}{n!}$$ $$\frac{e^x - e^{-x}}{2} = \sum_{n=0}^{\infty}\frac{x^{2n+1}}{(2n+1)!}$$ $$\frac{e^x + e^{-x}}{2} = \sum_{n=0}^{\infty}\frac{x^{2n}}{(2n)!}$$

第三组（三角函数）

$$\sin x = x - \frac{1}{6}x^3 + \cdots = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n+1}}{(2n+1)!}$$ $$\cos x = 1 - \frac{1}{2}x^2 + \frac{1}{24}x^4 + \cdots = \sum_{n=0}^{\infty}(-1)^n\frac{x^{2n}}{(2n)!}$$ $$\tan x = x + \frac{1}{3}x^3 + \frac{2}{15}x^5 + \cdots, \quad |x| < \frac{\pi}{2}$$

第四组（对数函数）

$$\ln(1+x) = x - \frac{1}{2}x^2 + \frac{1}{3}x^3 - \frac{1}{4}x^4 + \cdots = \sum_{n=1}^{\infty}(-1)^{n+1}\frac{x^n}{n}, \quad -1 < x \leq 1$$ $$\ln\frac{1+x}{1-x} = 2\left(x + \frac{1}{3}x^3 + \frac{1}{5}x^5 + \cdots\right) = 2\sum_{n=0}^{\infty}\frac{x^{2n+1}}{2n+1}, \quad |x| < 1$$
