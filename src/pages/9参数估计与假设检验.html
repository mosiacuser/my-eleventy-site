📍 一、求点估计、作评价（O₁(Ⅲ任性目标1)）

1. 矩估计 (D₁(常规操作))

对于一个参数：

a. 用一阶矩建立方程：令 $\widehat{X} = EX$；

b. 若 "a" 不能用，用二阶矩建立方程：令 $\frac{1}{n}\sum_{i=1}^{n}X_i^2 = E(X^2)$

一个方程解出一个参数即可作为矩估计。

对于两个参数：

用一阶矩与二阶矩建立两个方程，即 $\overline{X} = EX$ 与 $\frac{1}{n}\sum_{i=1}^{n}X_i^2 = E(X^2)$，两个方程解出两个参数即可作为矩估计。

🔍 例 9.1

设总体 $X$ 的概率密度为 $f(x) = \frac{1}{2\theta}e^{-\frac{|x|}{\theta}}$，$-\infty < x < +\infty$，$\theta > 0$。$X_1, X_2, \cdots, X_n$ 是取自总体 $X$ 的样本，则未知参数 $\theta$ 的矩估计量为________。

解：应填 $\sqrt{\frac{1}{2n}\sum_{i=1}^{n}X_i^2}$

总体只含有一个参数 $\theta$，但由于一阶矩建立的方程

$$EX = \int_{-\infty}^{+\infty} x \cdot \frac{1}{2\theta}e^{-\frac{|x|}{\theta}}dx = 0$$

无法解出 $\theta$，因此采用总体二阶矩建立方程：

$$E(X^2) = \int_{-\infty}^{+\infty} x^2 \cdot \frac{1}{2\theta}e^{-\frac{|x|}{\theta}}dx = 2 \cdot \frac{1}{2\theta}\int_{0}^{+\infty} x^2 \cdot e^{-\frac{x}{\theta}}dx$$ $$= \theta^2 \int_{0}^{+\infty} \frac{x^2}{\theta^2}e^{-\frac{x}{\theta}}d\left(\frac{x}{\theta}\right) \stackrel{t = \frac{x}{\theta}}{=} \theta^2\int_{0}^{+\infty} t^2e^{-t}dt = \theta^2\Gamma(3) = 2\theta^2$$

其中 $\Gamma(n+1) = \int_{0}^{+\infty} x^n e^{-x}dx = n!$。用样本二阶原点矩 $A_2 = \frac{1}{n}\sum_{i=1}^{n}X_i^2$ 代替总体二阶原点矩 $E(X^2)$ 得到 $\theta$ 的矩估计量为

$$\widehat{\theta} = \sqrt{\frac{A_2}{2}} = \sqrt{\frac{1}{2n}\sum_{i=1}^{n}X_i^2}$$

🔍 例 9.2

设总体 $X$ 服从含有两个参数的指数分布，其概率密度为

$$f(x;\theta,\lambda) = \begin{cases} \frac{1}{\lambda}e^{-\frac{1}{\lambda}(x-\theta)}, & x \geq \theta_0 (\lambda > 0) \\ 0, & \text{其他} \end{cases}$$

$X_1, X_2, \cdots, X_n$ 是来自总体 $X$ 的一个样本，求未知参数 $\lambda$、$\theta$ 的矩估计量。

解：这是求两个未知参数矩估计量的问题。由于

$$EX = \int_{\theta}^{+\infty} x \cdot \frac{1}{\lambda}e^{-\frac{1}{\lambda}(x-\theta)}dx \stackrel{\delta x - \theta = t}{=} \int_{0}^{+\infty} \frac{t+\theta}{\lambda}e^{-\frac{t}{\lambda}}dt$$ $$= ET + \theta = \lambda + \theta$$
$$E(X^2) = \int_{\theta}^{+\infty} x^2 \cdot \frac{1}{\lambda}e^{-\frac{1}{\lambda}(x-\theta)}dx \stackrel{\delta x - \theta = t}{=} \int_{0}^{+\infty} \frac{(t+\theta)^2}{\lambda}e^{-\frac{t}{\lambda}}dt$$ $$= E[(T+\theta)^2] = E(T^2) + 2\theta ET + \theta^2 = 2\lambda^2 + 2\theta\lambda + \theta^2$$

矩法方程为

$$\begin{cases} EX = \overline{X} \\ E(X^2) = \frac{1}{n}\sum_{i=1}^{n}X_i^2 \end{cases} \text{ 即 } \begin{cases} \lambda + \theta = \overline{X} \\ \lambda^2 + (\lambda + \theta)^2 = \frac{1}{n}\sum_{i=1}^{n}X_i^2 \end{cases}$$

由此解得 $\lambda$、$\theta$ 的矩估计量分别为

$$\widehat{\lambda} = \sqrt{\frac{1}{n}\sum_{i=1}^{n}X_i^2 - \overline{X}^2}, \quad \widehat{\theta} = \overline{X} - \sqrt{\frac{1}{n}\sum_{i=1}^{n}X_i^2 - \overline{X}^2}$$

其中 $\overline{X} = \frac{1}{n}\sum_{i=1}^{n}X_i$

2. 最大似然估计 (D₁(常规操作)+D₂₂(转换等价表述))

对未知参数 $\theta$ 进行估计时，在该参数可能的取值范围 $I$ 内选取，使 "样本获此观测值 $x_1, x_2, \cdots, x_n$" 的概率最大的参数值 $\widehat{\theta}$ 作为 $\theta$ 的估计，这样选定的 $\widehat{\theta}$ 最有利于 $x_1, x_2, \cdots, x_n$ 的出现，即 "参数 $\theta$ 为多少时，观测值出现的概率最大"。

若联合概率密度（连续型）或联合分布律（离散型）中有与参数 $\theta$ 无关的因式，为便于计算，去掉这些因式不会影响对 $\theta$ 的最大似然估计，比如 $P\{X_1 = k_1, X_2 = k_2, \cdots, X_n = k_n\} = n! \prod_{i=1}^{n} \frac{[P(\theta)]^{k_i}}{(k_i)!}$，可令 $L(\theta) = \prod_{i=1}^{n}[P(\theta)]^{k_i}$，但要注意，若求 $P\{X_1 = k_1, X_2 = k_2, \cdots, X_n = k_n\}$，则不能去掉任何因式。

求参数步骤：

  1. 若似然函数有驻点，则 $\frac{dL}{d\theta} = 0$ 或 $\frac{d(\ln L)}{d\theta} = 0$，解出 $\widehat{\theta}$；
  2. 若似然函数无驻点(单调)，则用定义求 $\widehat{\theta}$；
  3. 若似然函数为常数，则用定义求 $\widehat{\theta}$，此时 $\widehat{\theta}$ 不唯一。

③最大似然估计量的不变性原则：

设 $\widehat{\theta}$ 是总体分布中未知参数 $\theta$ 的最大似然估计，函数 $u = u(\theta)$ 具有单值反函数 $\theta = \theta(u)$，则 $\widehat{u} = u(\widehat{\theta})$ 是 $u(\theta)$ 的最大似然估计。

④双总体的最大似然估计。

🔍 例 9.3

设总体 $X$ 服从参数为 $\lambda$ ($\lambda > 0$ 但未知) 的泊松分布，$X_1, X_2, \cdots, X_n$ 是来自总体 $X$ 的一个简单随机样本，则 $P\{X = 0\}$ 的最大似然估计量为________。

解：应填 $e^{-\overline{X}}$

设 $X_1, X_2, \cdots, X_n$ 对应的样本值为 $x_1, x_2, \cdots, x_n$，则似然函数为

$$L(\lambda) = P\{X = x_1\}P\{X = x_2\}\cdots P\{X = x_n\}$$ $$= \frac{\lambda^{x_1}}{x_1!}e^{-\lambda} \cdot \frac{\lambda^{x_2}}{x_2!}e^{-\lambda} \cdot \ldots \cdot \frac{\lambda^{x_n}}{x_n!}e^{-\lambda} = \frac{e^{-n\lambda}}{x_1!x_2!\cdots x_n!}\lambda^{\sum_{i=1}^{n}x_i}$$

即

$$\ln L(\lambda) = -n\lambda - \sum_{i=1}^{n}\ln(x_i!) + \left(\sum_{i=1}^{n}x_i\right)\ln\lambda$$

令 $\frac{d[\ln L(\lambda)]}{d\lambda} = 0$，即 $-n + \frac{\sum_{i=1}^{n}x_i}{\lambda} = 0$，解得 $\widehat{\lambda} = \frac{1}{n}\sum_{i=1}^{n}x_i = \overline{x}$，即 $\lambda$ 的最大似然估计量为 $\widehat{\lambda} = \frac{1}{n}\sum_{i=1}^{n}X_i = \overline{X}$。代入

$P\{X = 0\} = \frac{\lambda^0}{0!}e^{-\lambda}$，由最大似然估计量的不变性原则，得 $P\{X = 0\}$ 的最大似然估计量为 $e^{-\overline{X}}$。

3. 估计量的评价（仅数学一）(D₁(常规操作)+D₂₃(化归经典形式))

（1）无偏性

对于估计量 $\widehat{\theta}$，若 $E\widehat{\theta} = \theta$，则称 $\widehat{\theta}$ 为 $\theta$ 的无偏估计量。

（2）有效性

若 $E\widehat{\theta}_1 = \theta$，$E\widehat{\theta}_2 = \theta$，即 $\widehat{\theta}_1, \widehat{\theta}_2$ 均是 $\theta$ 的无偏估计量，当 $D\widehat{\theta}_1 < D\widehat{\theta}_2$ 时，称 $\widehat{\theta}_1$ 比 $\widehat{\theta}_2$ 有效。

（3）一致性（相合性）（只针对大样本 $n \to \infty$）

若 $\widehat{\theta}$ 为 $\theta$ 的估计量，则对任意 $\varepsilon > 0$，有

$$\lim_{n\to\infty} P\{|\widehat{\theta} - \theta| \geq \varepsilon\} = 0$$

或

$$\lim_{n\to\infty} P\{|\widehat{\theta} - \theta| < \varepsilon\} = 1$$

即当 $\widehat{\theta} \stackrel{P}{\longrightarrow} \theta$ 时，称 $\widehat{\theta}$ 为 $\theta$ 的一致（或相合）估计。

一般利用切比雪夫不等式 $P\{|X - EX| \geq \varepsilon\} \leq \frac{DX}{\varepsilon^2}$

或常做大数定律 $P\{|X - EX| < \varepsilon\} = 1 - P\{|X - EX| \geq \varepsilon\}$

由 $\widehat{\theta} \stackrel{P}{\longrightarrow} EX$

4. 估计量的数字特征（仅数学三）(D₁(常规操作))

①求 $E\widehat{\theta}$

②求 $D\widehat{\theta}$

③验证 $\widehat{\theta}$ 是否依概率收敛于 $\theta$，即是否有 $\widehat{\theta} \stackrel{P}{\longrightarrow} \theta$，即对任意 $\varepsilon > 0$，有

$$\lim_{\theta\to\infty} P\{|\widehat{\theta} - \theta| \geq \varepsilon\} = 0 \text{ 或 } \lim_{n\to\infty} P\{|\widehat{\theta} - \theta| < \varepsilon\} = 1$$

🔍 例 9.7

设 $X_1, \cdots, X_n$ 独立同分布，$X_i$ 可取且只可取 4 个不同数值，且相应的取值概率分别为

$$p_1 = 1-\theta, p_2 = \theta - \theta^2, p_3 = \theta^2 - \theta^3, p_4 = \theta^3$$

记 $N_i (i=1,2,3,4)$ 为从总体抽取的 $n$ 个样本中出现 4 个不同数值所对应的个数。

(1)（仅数学一）确定 $a_1, a_2, a_3, a_4$ 使 $T = \sum_{i=1}^{4}a_iN_i$ 为 $\theta$ 的无偏估计；

（仅数学三）确定 $a_1, a_2, a_3, a_4$ 使 $E\left(\sum_{i=1}^{4}a_iN_i\right) = \theta$；

(2) 若 $N_i = n_i$，$i = 1,2,3,4$，求 $\theta$ 的最大似然估计值。

【解】(1)（仅数学一）由于 $N_i \sim B(n, p_i), i = 1,2,3,4$，因此 $EN_i = np_i$，从而有

$$ET = \sum_{i=1}^{4}a_iEN_i = a_1n(1-\theta) + a_2n(\theta - \theta^2) + a_3n(\theta^2 - \theta^3) + a_4n\theta^3$$ $$= na_1 + n(a_2 - a_1)\theta + n(a_3 - a_2)\theta^2 + n(a_4 - a_3)\theta^3$$

若使 $T$ 为 $\theta$ 的无偏估计，即要求

$$\begin{cases} na_1 = 0 \\ n(a_2 - a_1) = 1 \\ n(a_3 - a_2) = 0 \\ n(a_4 - a_3) = 0 \end{cases}$$

解得 $a_1 = 0, a_2 = a_3 = a_4 = \frac{1}{n}$

即 $T = \frac{N_2 + N_3 + N_4}{n}$ 是 $\theta$ 的无偏估计。

（仅数学三）由于 $N_i \sim B(n, p_i), i = 1,2,3,4$，因此 $EN_i = np_i$，从而有

$$E\left(\sum_{i=1}^{4}a_iN_i\right) = \sum_{i=1}^{4}a_iEN_i = a_1n(1-\theta) + a_2n(\theta - \theta^2) + a_3n(\theta^2 - \theta^3) + a_4n\theta^3$$ $$= na_1 + n(a_2 - a_1)\theta + n(a_3 - a_2)\theta^2 + n(a_4 - a_3)\theta^3$$

若使 $E\left(\sum_{i=1}^{4}a_iN_i\right) = \theta$，即要求

$$\begin{cases} na_1 = 0 \\ n(a_2 - a_1) = 1 \\ n(a_3 - a_2) = 0 \\ n(a_4 - a_3) = 0 \end{cases}$$

解得 $a_1 = 0, a_2 = a_3 = a_4 = \frac{1}{n}$

(2) 由题意，有 $N_1 + N_2 + N_3 + N_4 = n$，且似然函数为

$$L(\theta) = (1-\theta)^{n_1}(\theta - \theta^2)^{n_2}(\theta^2 - \theta^3)^{n_3}(\theta^3)^{n_4} = \theta^{n_2+2n_3+3n_4}(1-\theta)^{n_1+n_2+n_3}$$

取对数得

$$\ln L(\theta) = (n_2 + 2n_3 + 3n_4)\ln\theta + (n_1 + n_2 + n_3)\ln(1-\theta)$$

对 $\theta$ 求导，有

$$\frac{d[\ln L(\theta)]}{d\theta} = \frac{n_2 + 2n_3 + 3n_4}{\theta} - \frac{n_1 + n_2 + n_3}{1-\theta} \stackrel{\triangle}{=} 0$$

解得

$$\frac{1-\theta}{\theta} = \frac{n_1 + n_2 + n_3}{n_2 + 2n_3 + 3n_4}$$

则 $\theta$ 的最大似然估计值为

$$\widehat{\theta} = \frac{n_2 + 2n_3 + 3n_4}{n_1 + 2n_2 + 3n_3 + 3n_4}$$

📏 二、作区间估计、假设检验、求两类错误

1. 区间估计 (D₁(常规操作))

设 $\theta$ 是总体 $X$ 的分布函数的一个未知参数，对于给定的 $\alpha (0 < \alpha < 1)$，如果由样本 $X_1, X_2, \cdots, X_n$ 确定的两个统计量 $\widehat{\theta}_1 = \widehat{\theta}_1(X_1, X_2, \cdots, X_n)$，$\widehat{\theta}_2 = \widehat{\theta}_2(X_1, X_2, \cdots, X_n)$，使

$$P(\widehat{\theta}_1(X_1, X_2, \cdots, X_n) < \theta < \widehat{\theta}_2(X_1, X_2, \cdots, X_n)) = 1-\alpha$$

则称随机区间 $(\widehat{\theta}_1, \widehat{\theta}_2)$ 是 $\theta$ 的置信度为 $1-\alpha$ 的置信区间，$\widehat{\theta}_1$ 和 $\widehat{\theta}_2$ 分别称为 $\theta$ 的置信度为 $1-\alpha$ 的双侧置信区间的置信下限和置信上限。$1-\alpha$ 称为置信度或置信水平，$\alpha$ 称为显著性水平。如果 $P(\theta < \widehat{\theta}_1) = P(\theta > \widehat{\theta}_2) = \frac{\alpha}{2}$，则称这种置信区间为等尾置信区间。

设 $X \sim N(\mu, \sigma^2)$，从总体 $X$ 中抽取样本 $X_1, X_2, \cdots, X_n$，样本均值为 $\overline{X}$，样本方差为 $S^2$。

① $\sigma^2$ 已知，$\mu$ 的置信水平是 $1-\alpha$ 的置信区间为

$$\left(\overline{X} - \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}, \overline{X} + \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}\right)$$

记为 $I_1 \Rightarrow P\{\mu \in I_1\} = 1-\alpha$

② $\sigma^2$ 未知，$\mu$ 的置信水平是 $1-\alpha$ 的置信区间为

$$\left(\overline{X} - \frac{S}{\sqrt{n}}t_{\alpha}(n-1), \overline{X} + \frac{S}{\sqrt{n}}t_{\alpha}(n-1)\right)$$

记为 $I_2 \Rightarrow P\{\mu \in I_2\} = 1-\alpha$

③ $\mu$ 已知，$\sigma^2$ 的置信水平是 $1-\alpha$ 的置信区间为

$$\left(\frac{\sum_{i=1}^{n}(X_i - \mu)^2}{\chi_{\frac{\alpha}{2}}^2(n)}, \frac{\sum_{i=1}^{n}(X_i - \mu)^2}{\chi_{1-\frac{\alpha}{2}}^2(n)}\right)$$

记为 $I_3 \Rightarrow P\{\sigma^2 \in I_3\} = 1-\alpha$

此种情况一般不出现

④ $\mu$ 未知，$\sigma^2$ 的置信水平是 $1-\alpha$ 的置信区间为

$$\left(\frac{(n-1)S^2}{\chi_{\frac{\alpha}{2}}^2(n-1)}, \frac{(n-1)S^2}{\chi_{1-\frac{\alpha}{2}}^2(n-1)}\right)$$

记为 $I_4 \Rightarrow P\{\sigma^2 \in I_4\} = 1-\alpha$

🔍 例 9.9

设总体 $X$ 服从正态分布 $N(\mu, \sigma^2)$，其中 $\sigma^2$ 已知，$n$ 是给定的样本容量，$\mu$ 为未知参数，则 $\mu$ 的等尾双侧置信区间长度 $L$ 与置信度 $1-\alpha$ 的关系是（ ）。

(A) 当 $1-\alpha$ 减小时，$L$ 减小

(B) 当 $1-\alpha$ 减小时，$L$ 增大

(C) 当 $1-\alpha$ 减小时，$L$ 不变

(D) 当 $1-\alpha$ 减小时，$L$ 增减不定

【解】应选 (A)。

由均值 $\mu$ 的等尾双侧置信区间 $\left(\overline{X} - \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}, \overline{X} + \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}\right)$ 来确定正确选项。事实上，此时置信区间长度

$$L = \frac{2\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}$$

当 $1-\alpha$ 减小时，$\alpha$ 增大，$z_{\frac{\alpha}{2}}$ 减小，故 $L$ 减小，所以选择 (A)。

2. 假设检验 (D₁(常规操作))

关于总体（分布中的未知参数，分布的类型、特征、相关性、独立性……）的每一种论断（"看法"）称为统计假设，然后根据样本观测数据或试验结果提供的信息去推断（检验）这个"看法"（即假设）是否成立，这类统计推断问题称为假设检验。

常常把没有充分理由不能轻易否定的假设取为原假设（基本假设或零假设），记为 $H_0$，将其否定的陈述（假设）称为对立假设或备择假设，记为 $H_1$。

①小概率原理

对假设进行检验的基本思想是采用某种带有概率性质的反证法。这种方法的依据是小概率原理——概率很接近于 0 的事件在一次试验或观察中认为各备假设不会发生。若小概率事件发生了，则拒绝原假设。

②显著性水平 $\alpha$

小概率事件中"小概率"的值没有统一规定，通常是根据实际问题的要求，规定一个界限 $\alpha (0 < \alpha < 1)$，当一个事件的概率不大于 $\alpha$ 时，即认为它是小概率事件。在假设检验问题中，$\alpha$ 称为显著性水平，通常取 $\alpha = 0.1$，0.05，0.01 等。

① $\sigma^2$ 已知，$\mu$ 未知，$H_0: \mu = \mu_0$，$H_1: \mu \neq \mu_0$，则拒绝域为

$$\left(-\infty, \mu_0 - \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}\right) \cup \left[\mu_0 + \frac{\sigma}{\sqrt{n}}z_{\frac{\alpha}{2}}, +\infty\right)$$

② $\sigma^2$ 未知，$\mu$ 未知，$H_0: \mu = \mu_0$，$H_1: \mu \neq \mu_0$，则拒绝域为

$$\left(-\infty, \mu_0 - \frac{S}{\sqrt{n}}t_{\frac{\alpha}{2}}(n-1)\right) \cup \left[\mu_0 + \frac{S}{\sqrt{n}}t_{\frac{\alpha}{2}}(n-1), +\infty\right)$$

③ $\sigma^2$ 已知，$\mu$ 未知，$H_0: \mu \leq \mu_0$，$H_1: \mu > \mu_0$，则拒绝域为

$$\left[\mu_0 + \frac{\sigma}{\sqrt{n}}z_{\alpha}, +\infty\right)$$

(或若 $\mu = \mu_0$)

④ $\sigma^2$ 已知，$\mu$ 未知，$H_0: \mu \geq \mu_0$，$H_1: \mu < \mu_0$，则拒绝域为

$$\left(-\infty, \mu_0 - \frac{\sigma}{\sqrt{n}}z_{\alpha}\right]$$

(或若 $\mu = \mu_0$)

⑤ $\sigma^2$ 未知，$\mu$ 未知，$H_0: \mu \leq \mu_0$，$H_1: \mu > \mu_0$，则拒绝域为

$$\left[\mu_0 + \frac{S}{\sqrt{n}}t_{\alpha}(n-1), +\infty\right)$$

(或若 $\mu = \mu_0$)

⑥ $\sigma^2$ 未知，$\mu$ 未知，$H_0: \mu \geq \mu_0$，$H_1: \mu < \mu_0$，则拒绝域为

$$\left(-\infty, \mu_0 - \frac{S}{\sqrt{n}}t_{\alpha}(n-1)\right]$$

(或若 $\mu = \mu_0$)

(1) $H_0$ 中含 "="，以便 $H_0$ 成立时有分布可用。

(2) 拒绝域的 "形式" 与备择假设 $H_1$ 的 "含义" 一致，便于记忆。比如上述①中，$H_1: \mu \neq \mu_0$，则拒绝域为 "远离 $\mu_0$"，可见，"$\mu \neq \mu_0$" 与 "远离 $\mu_0$" 的含义一致。

🔍 例 9.11

设 $X_1, X_2, \cdots, X_n$ 为来自正态总体 $N(\mu, 2)$ 的简单随机样本，记 $\overline{X} = \frac{1}{n}\sum_{i=1}^{n}X_i z_{\alpha}$ 表示标准正态分布上 $\alpha$ 分位数。假设检验问题：$H_0: \mu \leq 1$，$H_1: \mu > 1$ 的显著性水平为 $\alpha$ 的检验的拒绝域为（ ）。

(A) $\left\{(X_1, X_2, \cdots, X_n) | \overline{X} \geq 1 + \frac{2}{n}z_{\alpha}\right\}$

(B) $\left\{(X_1, X_2, \cdots, X_n) | \overline{X} \geq 1 + \frac{\sqrt{2}}{n}z_{\alpha}\right\}$

(C) $\left\{(X_1, X_2, \cdots, X_n) | \overline{X} \geq 1 + \frac{2}{\sqrt{n}}z_{\alpha}\right\}$

(D) $\left\{(X_1, X_2, \cdots, X_n) | \overline{X} \geq 1 + \sqrt{\frac{2}{n}}z_{\alpha}\right\}$

【解】应选 (D)。

由题意知，$\sigma = \sqrt{2}$，且符合上述六大情形的③，故拒绝域为 $\overline{X} \geq \mu_0 + \frac{\sigma}{\sqrt{n}}z_{\alpha} = 1 + \sqrt{\frac{2}{n}}z_{\alpha}$，选 (D)。

3. 两类错误 (D₁(常规操作))

第一类错误（"弃真"）：若 $H_0$ 为真，按检验法则，否定了 $H_0$，此时犯了 "弃真" 的错误，这种错误称为第一类错误，犯第一类错误的概率为 $\alpha = P\{$ 拒绝 $H_0 | H_0$ 为真$\}$。

第二类错误（"取伪"）：若 $H_0$ 不真，按检验法则，接受 $H_0$，此时犯了 "取伪" 的错误，这种错误称为第二类错误，犯第二类错误的概率为 $\beta = P\{$ 接受 $H_0 | H_0$ 为假$\}$。

🔍 例 9.12

设 $X_1, X_2, \cdots, X_{16}$ 是来自总体 $N(\mu, 4)$ 的简单随机样本，考虑假设检验问题：$H_0: \mu \leq 10$，$H_1: \mu > 10$。$\Phi(x)$ 表示标准正态分布函数。若该检验问题的拒绝域为 $W = \{\overline{X} > 11\}$，其中 $\overline{X} = \frac{1}{16}\sum_{i=1}^{16}X_i$，则

$\mu = 11.5$ 时，该检验犯第二类错误的概率为（ ）。

(A) $1-\Phi(1)$    (B) $1-\Phi(0.5)$    (C) $1-\Phi(1.5)$    (D) $1-\Phi(2)$

【解】应选 (A)。

当 $\mu = 11.5$ 时，该检验犯第二类错误的概率为

$$P\{\overline{X} \leq 11 | \mu = 11.5\}$$

当 $\mu = 11.5$ 时，

$$\overline{X} \sim N\left(11.5, \frac{4}{16}\right), \text{ 即 } \overline{X} \sim N\left(11.5, \frac{1}{4}\right)$$

于是

$$Z = \frac{\overline{X} - \mu}{1/\sqrt{36}} \stackrel{\text{近似}}{\sim} N(0, 1)$$

从而

$$P\{\overline{X} \leq 11 | \mu = 11.5\} = \Phi\left(\frac{11-11.5}{1/2}\right) = \Phi(-1) = 1-\Phi(1)$$

🔍 例 9.4

设总体 $X$ 的概率分布为 $P\{X = 1\} = \frac{1-\theta}{2}$，$P\{X = 2\} = P\{X = 3\} = \frac{1+\theta}{4}$，利用来自总体 $X$ 的样本值 1, 3, 2, 2, 1, 3, 1, 2，可得 $\theta$ 的最大似然估计值为（ ）。

(A) $\frac{3}{8}$    (B) $\frac{1}{4}$    (C) $\frac{1}{2}$    (D) $\frac{5}{8}$

【解】应选 (B)。

由题意可得似然函数为

$$L(\theta) = \left(\frac{1-\theta}{2}\right)^3 \cdot \left(\frac{1+\theta}{4}\right)^5$$

两边取对数，得

$$\ln L(\theta) = 3\ln\frac{1-\theta}{2} + 5\ln\frac{1+\theta}{4} = 3\ln(1-\theta) - 3\ln 2 + 5\ln(1+\theta) - 5\ln 4$$

令

$$\frac{d[\ln L(\theta)]}{d\theta} = \frac{-3}{1-\theta} + \frac{5}{1+\theta} = 0$$

得 $\theta$ 的最大似然估计值为 $\widehat{\theta} = \frac{1}{4}$，故选 (B)。

🔍 例 9.5

设总体 $X$ 的概率密度为 $f(x) = \begin{cases} \frac{2x}{\alpha^2}, & 0 \leq x \leq \alpha \\ 0, & \text{其他} \end{cases}$，其中 $\alpha (\alpha > 1)$ 是未知参数，$X_1, X_2, \cdots, X_n$ 是来自总体 $X$ 的简单随机样本，$X_{(1)} = \min\{X_1, X_2, \cdots, X_n\}$，$X_{(n)} = \max\{X_1, X_2, \cdots, X_n\}$，则 $p$ 的最大似然估计量 $\widehat{p}$ 为（ ）。

(A) $\sqrt{X_{(1)}}$    (B) $\sqrt{X_{(n)}}$    (C) $\frac{1}{X_{(1)}}$    (D) $\frac{1}{X_{(n)}}$

【解】应选 (D)。

$$p = P\{0 < X < \sqrt{\alpha}\} = \int_{0}^{\sqrt{\alpha}} f(x)dx = \frac{1}{\alpha}$$

当 $0 \leq x_1 \leq \alpha, 0 \leq x_2 \leq \alpha, \cdots, 0 \leq x_n \leq \alpha$ 时，似然函数为

$$L(\alpha) = f(x_1)f(x_2)\cdots f(x_n) = \frac{2^n}{\alpha^{2n}}x_1x_2\cdots x_n$$

显然 $L(\alpha)$ 关于 $\alpha$ 单调递减少，且 $\alpha \geq \max\{x_1, x_2, \cdots, x_n\}$，则 $\alpha$ 的最大似然估计量为

$$\widehat{\alpha} = \max\{X_1, X_2, \cdots, X_n\}$$

又知 $p = \frac{1}{\alpha}$ 是关于 $\alpha$ 的单调函数，根据最大似然估计量的不变性原则，得 $p$ 的最大似然估计量为

$$\widehat{p} = \frac{1}{\max\{X_1, X_2, \cdots, X_n\}} = \frac{1}{X_{(n)}}$$

🔍 例 9.6

设 $X_1, X_2, \cdots, X_n$ 为来自均值为 $\theta$ 的指数分布总体的简单随机样本，$Y_1, Y_2, \cdots, Y_m$ 为来自均值为 $2\theta$ 的指数分布总体的简单随机样本，且两样本相互独立，其中 $\theta (> 0)$ 是未知参数，利用样本 $X_1, X_2, \cdots, X_n, Y_1, Y_2, \cdots, Y_m$ 求 $\theta$ 的最大似然估计量 $\widehat{\theta}$，并求 $D\widehat{\theta}$。

【解】设 $x_1, x_2, \cdots, x_n, y_1, y_2, \cdots, y_m$ 为样本值，则似然函数为

$$L(\theta) = \frac{1}{2^m \theta^{m+n}}e^{-\frac{1}{\theta}\sum_{i=1}^{n}x_i - \frac{1}{2\theta}\sum_{j=1}^{m}y_j}$$

从而

$$\ln L(\theta) = -m\ln 2 - (n+m)\ln\theta - \frac{1}{\theta}\sum_{i=1}^{n}x_i - \frac{1}{2\theta}\sum_{j=1}^{m}y_j$$
$$\frac{d[\ln L(\theta)]}{d\theta} = -\frac{n+m}{\theta} + \frac{1}{\theta^2}\sum_{i=1}^{n}x_i + \frac{1}{2\theta^2}\sum_{j=1}^{m}y_j$$

令

$$\frac{d[\ln L(\theta)]}{d\theta} = 0$$

解得

$$\theta = \frac{1}{n+m}\left(\sum_{i=1}^{n}x_i + \frac{1}{2}\sum_{j=1}^{m}y_j\right)$$

因此 $\theta$ 的最大似然估计量为

$$\widehat{\theta} = \frac{2n\overline{X} + m\overline{Y}}{2(n+m)}$$

其中 $\overline{X} = \frac{1}{n}\sum_{i=1}^{n}X_i$，$\overline{Y} = \frac{1}{m}\sum_{j=1}^{m}Y_j$

由于 $D\overline{X} = \frac{\theta^2}{n}$，$D\overline{Y} = \frac{4\theta^2}{m}$，所以

$$D\widehat{\theta} = D\left[\frac{2n\overline{X} + m\overline{Y}}{2(n+m)}\right] = \frac{4n^2D\overline{X} + m^2D\overline{Y}}{4(n+m)^2} = \frac{\theta^2}{n+m}$$

🔍 例 9.8

设总体 $X$ 的分布函数为

$$F(x;\theta) = \begin{cases} 1-e^{-\frac{x^2}{\theta}}, & x > 0 \\ 0, & x < 0 \end{cases}$$

其中 $\theta$ 是未知参数且大于零，$X_1, X_2, \cdots, X_n$ 为来自总体 $X$ 的简单随机样本。

(1) 求 $EX$ 与 $E(X^2)$；

(2) 求 $\theta$ 的最大似然估计量 $\widehat{\theta}$；

(3) 是否存在实数 $a$，使得对任何 $\varepsilon > 0$，都有 $\lim_{n\to\infty} P\{|\widehat{\theta} - a| \geq \varepsilon\} = 0$？

【解】(1) 总体 $X$ 的概率密度为

$$f(x;\theta) = \begin{cases} \frac{2x}{\theta}e^{-\frac{x^2}{\theta}}, & x > 0 \\ 0, & \text{其他} \end{cases}$$
$$EX = \int_{0}^{+\infty} x \cdot \frac{2x}{\theta}e^{-\frac{x^2}{\theta}}dx = \int_{0}^{+\infty} x d\left(e^{-\frac{x^2}{\theta}}\right) = \int_{0}^{+\infty} e^{-\frac{x^2}{\theta}}dx = \frac{\sqrt{\pi\theta}}{2} \cdot \frac{1}{\sqrt{\pi\theta}}\int_{-\infty}^{+\infty} e^{-\frac{x^2}{\theta}}dx = \frac{\sqrt{\pi\theta}}{2}$$
$$E(X^2) = \int_{0}^{+\infty} x^2 \cdot \frac{2x}{\theta}e^{-\frac{x^2}{\theta}}dx = \theta\int_{0}^{+\infty} ue^{-u}du = \theta$$

(2) 设 $x_1, x_2, \cdots, x_n$ 为样本值，则似然函数为

$$L(\theta) = \prod_{i=1}^{n}f(x_i;\theta) = \begin{cases} \frac{2^n x_1x_2\cdots x_n}{\theta^n}\exp\left\{-\frac{1}{\theta}\sum_{i=1}^{n}x_i^2\right\}, & x_1, x_2, \cdots, x_n > 0 \\ 0, & \text{其他} \end{cases}$$

当 $x_1, x_2, \cdots, x_n > 0$ 时，$\ln L(\theta) = n\ln 2 + \sum_{i=1}^{n}\ln x_i - n\ln\theta - \frac{1}{\theta}\sum_{i=1}^{n}x_i^2$，令

$$\frac{d[\ln L(\theta)]}{d\theta} = -\frac{n}{\theta} + \frac{1}{\theta^2}\sum_{i=1}^{n}x_i^2 = 0$$

得

$$\theta = \frac{1}{n}\sum_{i=1}^{n}x_i^2$$

从而 $\theta$ 的最大似然估计量为 $\widehat{\theta} = \frac{1}{n}\sum_{i=1}^{n}X_i^2$

(3) 存在，$a = \theta$。因为 $\{X_i^2\}$ 是独立同分布的随机变量序列，且 $E(X_i^2) = \theta < +\infty$，所以根据大数定律，当 $n \to \infty$ 时，$\widehat{\theta} = \frac{1}{n}\sum_{i=1}^{n}X_i^2$ 依概率收敛于 $E(X_i^2)$，即 $\theta$，所以对任何 $\varepsilon > 0$，都有

$$\lim_{\theta\to\infty} P\{|\widehat{\theta} - \theta| \geq \varepsilon\} = 0$$

🔍 例 9.10

设总体 $X$ 的数学期望存在且方差为 1，根据来自 $X$ 的容量为 36 的简单随机样本测得样本均值为 $a$，$\Phi(1.96) = 0.975$，则 $X$ 的数学期望的置信度等于 0.95 的置信区间为（ ）。

(A) $(a - 0.196, a + 0.196)$

(B) $(a - 0.327, a + 0.327)$

(C) $(a - 0.49, a + 0.49)$

(D) $(a - 0.025, a + 0.025)$

【解】应选 (B)。

由中心极限定理，若 $X_i$ 满足定理条件，则当 $n \to \infty$ 时，$\sum_{i=1}^{n}X_i$ 近似服从 $N(n\mu, n\sigma^2)$，故

$$\sum_{i=1}^{36}X_i \stackrel{\text{近似}}{\sim} N(36\mu, 36), \overline{X} = \frac{1}{36}\sum_{i=1}^{36}X_i \stackrel{\text{近似}}{\sim} N\left(\mu, \frac{1}{36}\right)$$

于是

$$Z = \frac{\overline{X} - \mu}{1/\sqrt{36}} \stackrel{\text{近似}}{\sim} N(0,1)$$

又由 $\Phi(1.96) = 0.975$，故 $P\{-1.96 < Z < 1.96\} = 0.95$，即 $\mu$ 的置信度等于 0.95 的置信区间为

$$\left(\overline{X} - \frac{1}{6}\times 1.96, \overline{X} + \frac{1}{6}\times 1.96\right)$$

将 $\overline{x} = a$ 代入上式得置信区间为 $(a - 0.327, a + 0.327)$。
